# 04. 새 세션 부트스트랩

1. 권위 README와 01–03을 읽는다.
2. 대상 작품 source_lock과 work_state를 읽는다.
3. 원본 대본이 없으면 분석하지 않고 `SOURCE_MISSING`으로 기록한다.
4. work_state.next_action부터 재개하고 완료 회차를 임의 재생성하지 않는다.
5. 현재 블록 대본을 직접 읽고 Stage01→02→03을 저작한다.
6. 블록 감사 PASS 후 다음 최대 8회 블록으로 이동한다.
7. 전체 블록 완료 후 Stage04를 전 시즌 기준으로 신규 저작한다.
8. 네 층 PASS를 분리 보고한다.

금지: 과거 작품 복제, 고정 수량 목표, 자동 PASS의 의미 PASS 확대, Stage04 단순 합본, SourceLock 없는 PASS 선언.

최소 시작 명령문: “DRAMA_ANALYSIS_SINGLE_AUTHORITY_V7을 유일 권위로 사용한다. 원본 대본을 직접 독해하고 전체 회차를 최대 8회 연속 블록으로 나누어 순서대로 Stage01→02→03을 저작·감사한 뒤, 전 블록 완료 후 Stage04를 전 시즌 기준으로 저작한다. 네 층 PASS를 분리 보고한다.”
