# 재가중 held-out 교차검증 — 이전 주장 교정 (2026-06-07)

직전 'fitness 재가중 0.40→0.59 개선' 주장을 **작품 단위 Leave-One-Out 교차검증**(누수 없음)으로 엄밀 검증.

## 방법
5편 중 4편으로 상관기반 가중치 학습 → 나머지 1편(held-out)에 적용. 5폴드 풀링(30 held-out 씬).

## 결과 (held-out 30씬)
- 기존 가중치 fitness vs 품질: Spearman = **+0.698**
- 상관기반 재가중 fitness vs 품질: Spearman = **+0.700**
- 일반화 효과(차) = **+0.002** (사실상 0)
- (참고) in-sample: 0.70 / 0.73

## 교정·결론 (정직)
1. **이전 '재가중 0.40→0.59' 개선은 과적합 아티팩트.** 같은 데이터에 fit+eval한 결과였고, held-out에선 효과 0(+0.002). 이전 followup 문서의 해당 주장 철회.
2. **기존 fitness 가중치는 held-out에서도 0.70으로 잘 일반화** — 현재 공식 가중치가 (이 프록시 척도에서) 깨져 있지 않다. 앞선 N=18 '종합0.40<motif0.82'는 소표본 노이즈였음.
3. 단, 절대값 0.70은 proxy GT(LLM 품질)·동일LLM 순환으로 낙관적. **상대 결론(재가중≈기존)은 프록시에 견고**.

## 의의
held-out 검증이 내 이전 과대주장을 잡아냈다. = 공식 검증 하니스가 '개선처럼 보이는 과적합'을 걸러내는 안전장치로 작동.
