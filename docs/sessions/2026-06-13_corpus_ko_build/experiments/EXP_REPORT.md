# 공식 검증 1차 실험 리포트 (2026-06-13)

**데이터**: corpus_ko 106편(9,744씬, ChromaDB+SceneFeature+NKG). 전부 쌍대(pairwise)·순환성 회피 원칙 준수.
**메타-GT**: 전문가(수상·평단 1~4) + 관객(관객수 1~4), 63편에 **잠정** 부여(내 학습지식 ~2025). **개발자 권위 데이터로 교체 필요** — 본 수치는 방법론 실증용.
**스크립트**: `experiments/{meta_gt,run,exp_cbd,exp_d}.py` (멱등).

## ★EXP-A FE-7 — 공식점수 vs 메타-GT (n=62, 쌍대)

| 비교 | 일치율(concordance) | Kendall τ |
|---|---|---|
| 등가중 fitness vs 전문가 | 0.516 | +0.03 |
| 등가중 fitness vs 관객 | 0.433 | −0.13 |
| 등가중 fitness vs 전문가+관객 | 0.468 | −0.06 |

**소견**: 현재 등가중 fitness는 명성과 거의 무상관(≈우연). 단, **성분 분해 시 신호가 분리됨**:

| 성분 | τ vs (전문가+관객) |
|---|---|
| conflict_arc(갈등 동적범위) | **+0.17** |
| conflict_mean(갈등 밀도) | **+0.16** |
| curiosity_mean | +0.02 |
| motif_mean | −0.15 |
| energy_mean | −0.15 |
| climax_build | −0.18 |

→ **갈등 관련 신호는 명성과 정렬, energy/motif/climax는 역상관.** 관객(흥행)은 구조적 밀도와 음의 상관(블록버스터≠구조적으로 조밀).

## ★EXP-D 공식↔Critic 학습루프 1차 (정직한 LOO-CV)

| 방식 | concordance |
|---|---|
| 등가중 baseline | 0.458 |
| 적합(in-sample) | 0.664 |
| **LOO-CV(out-of-sample)** | **0.640** |

**핵심**: 선형 재가중이 **표본외에서도 0.64로 일반화**(과적합 아님; in-sample 0.664→CV 0.640 소폭 하락). 즉 **공식은 인간 메타-GT에 보정 가능**하다는 1차 실증. 학습 가중치 부호: conflict +, energy/motif/climax −. (과거 소표본 CV는 +0.002로 과적합이었으나, 62편 규모에서 일반화로 전환.)

## EXP-C F-24/25 — 장르별 긴장곡선 (scene_energy, 20분位)

장르군 평균곡선이 **형태상 뚜렷이 구분**(평균 쌍대 L1 거리 0.286 ≫ 0.1):

| 장르 | 피크 위치 | 형태 |
|---|---|---|
| thriller(11) | bin7(전반) | 초반 긴장→지속적 저강도 |
| romance(4) | bin8 | 중반 피크→결말 급강하(0.08) |
| melo(3) | bin6 | 중반 정점(0.91)→해소 |
| crime(6)/drama(11) | bin11-12 | 후반 상승(end 0.62) |
| period(5) | bin11 | 중후반 정점 |
| comedy(5) | bin15(후반) | 후반 빌드업 |
| action(5) | bin12 | 클라이맥스 후 급강하 |

→ **T_ideal은 장르 분리 필수**라는 F-24/25 전제를 실증.

## EXP-B DRSE 잔향 정렬 — 정직한 약결과

motif_residue vs 위치 상관 평균 **0.009**(55.8%만 후반 잔향 누적). 현재 잔향 지표(이전 씬 평균 코사인)는 클라이맥스 콜백 구조를 **코퍼스 전반에서 약하게만** 포착. 단 주제밀도 높은 작품(지구를지켜라 0.60·JSA 0.45·박쥐 0.24)은 정상 상위 → 신호는 존재하나 정의 개선 필요(평균유사 → 특정 모티프 재등장 측정).
