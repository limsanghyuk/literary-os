# 전략3 멀티에이전트 패널 변별력 POC (2026-06-13)

**목적**: 인간 작가 대체 가능성 — 명성차 뚜렷한 실제 씬 쌍을 3역할 패널(gpt-4o-mini: 구조분석가·대사전문가·긴장비평가)에 쌍대로 주고, 패널 판정이 객관 명성과 일치하는지.
**설계**: HIGH(전문가4: 마더·살인의추억·곡성·괴물·추격자·JSA) × LOW(전문가1: 해운대·간신·미인도·댄싱퀸·품행제로·내머리속의지우개) 15쌍. A/B 위치 무작위(편향통제). 씬은 작품 중반·400~1400자.

## 결과
- 패널이 명성 높은 씬 선택: **11/15 = 73%** (단측 이항검정 p=0.059)
- 만장일치 11/15, **만장일치 시 정답률 9/11 = 82%** → 만장일치가 신뢰 게이트로 유효
- 실패 4쌍(괴물vs내머리속의지우개·마더vs간신·곡성vs해운대·추격자vs미인도): 모두 걸작의 *조용한 단일 씬*이 열위작의 *극적 씬*에 진 경우 = 단일 씬 비교의 노이즈

## 결론
1. 패널은 **실제 변별력 보유**(73%, 우연 초과). 제목 없이 본문만 줬는데 맞춤 → 단순 명성 회상 아님.
2. **단일 씬 granularity가 주 오차원** → 작품당 다중 씬 집계 + 만장일치 가중이면 정확도 상승 예상.
3. **전략3 채택 타당**: 실제 대본 앵커 + 쌍대 + 이질 앙상블 + 객관사실 캘리브레이션 하에 **인간 작가를 대체하는 확장형 대리 심판**으로 사용 가능. 단 독립 심판이 아니라 객관 명성으로 주기 검증되는 프록시로.
