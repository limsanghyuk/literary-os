# 제안서 — NKG 인물 추출(시리즈 단위 NER) 보강 (2026-06-13)

**대상 결함**: 코퍼스 395편 중 166편(주로 드라마) NKG 인물 NOCHAR + 일부 장소명 누출('거실·안방·상무실'). NKG = tri-store 3레그 중 인물/관계 레그. (벡터·피처 레그는 100% 정합 — 본 보강은 NKG 한정.)

## 1. 근본 원인 진단 (4차원)
1. **입자도**: 드라마 인물은 *시리즈 전체* 재등장인데 회차별 추출은 재등장 기준 미달.
2. **포맷 이질성**: 대사 큐가 5형식(이름: / 단독행 / 이름⇥대사 / 이름+대사 붙음 / 지문 내).
3. **장소 누출**: 슬러그/헤딩 라인('거실','상무실')이 2-4자 이름 패턴과 충돌 → 인물로 오인.
4. **재등장 임계 부재**: 1회성 단역·오타가 캐스트에 혼입.

## 2. 설계 — 시리즈 단위 다중신호 NER

**핵심 원리**: *화자 위치는 인물의 강한 신호, 장소는 헤딩에만 출현* → 둘을 분리.

| 단계 | 방법 |
|---|---|
| ① 시리즈 풀링 | work_id 접두→시리즈 매핑, 회차 묶음 |
| ② 화자 후보 | **화자 위치만** 수집: `이름:대사` · `이름⇥대사` · `단독행 이름+다음행 대사`. (헤딩/지문 제외) |
| ③ 장소 제외 | 시리즈 전 헤딩 토큰=장소어휘 사전 구축 → 후보의 헤딩빈도 > 화자빈도면 탈락. **+ 접미 규칙(실·방·관·집·청·국·실내·실외) 장소 강제 제외** |
| ④ 재등장 임계 | 화자로 **≥2 회차** 등장한 후보만 캐스트 |
| ⑤ 형태소 보강 | mecab NNP(고유명사) 가중 ↑, NNG(일반명사=장소/직책) 가중 ↓ |
| ⑥ 친족어 처리 | '아버지·할머니' 등은 역할인물로 별도 태그(person_role) |
| ⑦ LLM 폴백 | 위로도 빈약한 시리즈만 **시리즈당 1콜**(24콜, 회차별 191콜 아님): 풀링 후보리스트 확인·필터 |
| ⑧ NKG 엣지 | 확정 캐스트의 씬 내 화자 인접 → 관계 엣지·공기쌍 |

## 3. 프로토타입 실증 (`char_ner_proto.py`, 화자한정+헤딩제외)
| 시리즈 | 추출 캐스트(상위) | 평가 |
|---|---|---|
| 원더풀라이프 | 일진·세진·채영·승완·한범수·윤태희 | ✅ 깨끗 |
| 역전의여왕 | 태희·준수·여진·용식 (+상무실·팀장실 잔여) | △ -실 접미 필터 필요 |
| 옥탑방고양이 | 동준·정우·혜련·준호·정민(+아버지/할머니) | ✅ (친족어 태그 필요) |
| 궁 | [] | ✗ 포맷 특이 → LLM 폴백 대상 |

**검증된 것**: naive 시리즈추출('거실·안방' 누출) → 화자한정+헤딩제외로 **실인물 다수 정확 추출, 장소 대폭 감소**. **남은 결함**: ③의 -실/-방 접미 규칙 미적용, 궁 등 특이포맷, 일부 시리즈 under-grouping(매핑 버그).

## 4. 보강 작업 명세 (DoD)
1. `corpus_ko/char_ner.py` — 위 ①~⑧ 구현. ③에 **접미 장소사전 + mecab NNP/NNG** 추가.
2. 매핑 버그 수정(별순검S1·개와늑대 under-grouping 원인=work_id 접두 정규화).
3. LLM 폴백: NOCHAR/빈약 시리즈만 시리즈당 1콜.
4. nkg.json 재생성 + 검증: **NOCHAR ≈ 0(드라마), 장소누출 0**, 캐스트 스팟체크(궁→신/채경, 옥탑방→정은/경민 등 알려진 배역 일치).
5. NKG 관계 엣지 재산출.

## 5. 다른 보강 필요 항목 (함께 정리)
- **드라마 메타GT(시청률·수상)**: 알고리즘 아닌 *데이터 취합* 문제. 공개사실(닐슨/TNMS 시청률·백상예술대상)로 가능 → 드라마 FE-7 개통.
- **씬 craft 피처**(루프B 재생 조건): 표면통계→craft상관 신호(임베딩 품질·서브텍스트·LLM태그) 연구과제.
- **이미지PDF OCR·rar 추출**: 도구(한국어 tesseract·unrar) 확보 문제.

## 6. 우선순위 판단
NKG 인물 NER 보강은 **중간 우선**: 구조 실험(장르곡선·DRSE·FE-7·패널)은 인물 불요로 이미 가동 중이라 급하지 않음. 단 **생성 Pass3/4(씬브리프 등장인물·NKG 관계 주입)**의 입력이므로, **생성 본체 구현(P4) 전에는 완료 필요**. 따라서 권장 순서: 생성 Pass4~7 착수와 **병행**해 본 NER 보강(소넷 집행, ~반나절 규모).
