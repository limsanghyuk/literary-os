# NER LLM 폴백 — NOCHAR 75편 해소, 인물 커버리지 100% (2026-06-16)

**계기**: 인코딩 교정·재처리 후에도 휴리스틱 char_ner가 인물 추출 실패한 75편(NOCHAR) 잔존 — 주로 시리즈 포맷 특이(궁01-23, 넌어느별에서왔니, 적도의남자 등). 다음 우선순위 ① 실행.

## 방법 (`llm_chars_run.py`)
- gpt-4o-mini, temperature=0, 본문 샘플(앞 6k + 중간 4k)에서 주요 인물만 JSON 배열 추출.
- **환각 방지 grounding**: 추출 이름 중 **본문 출현 ≥3회 + 길이 2~6자**만 채택, 출현빈도순 상위 12명.
- idempotent(char_method=="llm" 스킵) + 36초 버스트 재개 → 5회 버스트로 완주.

## 결과
- **NOCHAR 75 → 0. 인물 커버리지 100%** (전 455작품).
- 방식 구성: LLM폴백 **75** + 휴리스틱 **380**. 총 인물 노드 **5,889** (평균 12.9 / median 15명).
- 추출 품질(스팟체크): 궁=채경·황후·황태후·황제·효린 / 부산행=석우·수안·상화·성경 / 건축학개론=승민·서연·은채 — **주연 정확**.

## 한계 (정직)
- grounding은 빈도 필터일 뿐 의미 검증 아님 → '아빠·엄마' 같은 역할어가 일부 포함(한국 드라마 맥락상 유효 캐릭터라 허용).
- 시리즈 에피소드별 독립 호출(궁01~23 각각) → 동일 인물의 표기 흔들림 가능. 시리즈 정규화(union+표기통일)는 후속 과제.
- LLM 추출이라 LLM-1 경계 내(critic/orchestration 외 corpus 보조). corpus 본체 공식·constitution엔 미관여(G_LLM1_BOUNDARY 준수).
