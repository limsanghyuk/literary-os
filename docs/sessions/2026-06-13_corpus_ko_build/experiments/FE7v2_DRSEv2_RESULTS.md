# 권위 메타GT FE-7 재실행 + DRSE 모티프-사전 재설계 결과 (2026-06-13)

## (a) 권위 메타GT FE-7 v2

**메타GT v2**(`meta_gt_v2.py`): 잠정 1-4티어 → **실제 전국 관객수(연속, KOBIS 지식)** + **수상 기반 전문가 0~4**. 56편. *공개사실 기반(KOBIS·수상DB 검증가능), 2004 이전 전국 추정.*
**fitness_v2** = conflict_mean + conflict_arc + thematic_complexity + curiosity (③ 재정의 셋, 등가중).

| fitness_v2 vs | concordance | Kendall τ |
|---|---|---|
| 관객수(연속) | 0.580 | +0.16 |
| 전문가(수상) | 0.543 | +0.085 |
| **관객+전문가(z합)** | **0.598** | **+0.196** |

(이전 잠정 1-4티어·등가중 fitness: τ −0.06 → **권위 GT+재정의로 +0.196 전환**)

**성분별 τ (vs 관객+전문가)**: thematic_complexity **+0.195**(최강) · conflict_arc +0.182 · conflict_mean +0.147 · curiosity +0.053.

**학습루프 LOO-CV(v2 GT)**: 등가중 0.598 = 적합 0.598 → **LOO-CV 0.594** (과적합 0, 등가중이 이미 최적).

**소견(정직)**:
1. 권위 GT가 신호를 또렷하게 — fitness_v2는 명성과 τ≈0.20, concordance 0.60으로 우연(0.5) 명확 초과.
2. **thematic_complexity가 최강 신호**로 확정(③ 재정의 검증). conflict 계열 견고. curiosity 약함.
3. **관객수(0.16) > 수상(0.085)** — 공식은 비평적 명성보다 **상업적 몰입**을 더 포착.
4. 천장 ~0.60 < 패널 0.73 재확인 → 공식=baseline, 패널=고정밀 보상(하이브리드 정당).

## (b) DRSE 모티프-사전 재설계

**재설계**(`motif_drse.py`): 임베딩 코사인 폐기 → **작품별 재등장 모티프 사전**(씬 3회 이상·전체 50% 미만 출현 토큰; 한글 2-4자 정규식 토큰화, 불용어 제거). 잔향[k] = 씬 k의 모티프 중 **이전 씬에서 이미 등장한 것의 비율**(콜백률).

| 지표 | 이전(임베딩) | **DRSE v2(사전)** |
|---|---|---|
| 잔향 vs 위치 상관 평균 | 0.009 | **0.761** |
| 후반 콜백 누적 작품 비율 | 55.8% | **100%** |

최고 콜백누적: 지구를지켜라(0.97)·신세계(0.88)·끝까지간다·태후16 — 모티프 회수 강한 작품으로 타당.

**소견(정직)**:
1. **잔향 측정 자체는 해결** — 명시적 재등장 모티프 추적이 임베딩 유사보다 압도적(0.009→0.761). 이야기 진행에 따른 콜백 누적 곡선을 제대로 포착.
2. 단 **모티프 양(recurrence/richness)은 명성 신호 아님**(τ −0.04 / +0.07) — 잔향은 *구조 모델링*용이지 품질판별용이 아님(정상). 품질은 conflict·thematic이 담당.
3. 한계: konlpy 부재로 정규식 토큰화(조사 분리 불완전) → mecab 형태소 분석 시 모티프 사전 정밀도 상승. 또 콜백 *양*보다 *타이밍/배치*가 품질과 연관일 가능성(다음 정련).

## 종합
- FE-7은 권위 GT에서 유의미 정렬 확정(τ0.20), thematic_complexity가 핵심.
- DRSE 잔향은 모티프-사전으로 근본 수정 성공(0.76) — F-07~10 구조 신호로 사용 가능.
