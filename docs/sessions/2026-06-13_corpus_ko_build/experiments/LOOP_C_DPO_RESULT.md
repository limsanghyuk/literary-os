## 2. 정직한 confound (해석 주의)
"생성 59% 승"은 **생성>명작이 아님**. 교란 3종:
1. **LLM 심판 산문편향**(longform 기존진단): gpt-4o-mini는 유창한 산문 → 심판이 선호.
2. **레퍼런스=임의 명작 씬**(명작의 *평범한* 전환씬 포함, 베스트 아님).
3. 단일 씬·n=17 노이즈.
→ e2e_pass5_local(Pass4 RAG+구조브리프, seed고정)는 2/4=50%, 본 경량판(RAG·구조 없음)은 59% — 노이즈 범위. **진짜 loop-C 격차는 ① 명작 베스트씬 ② 인간 GT ③ 산문편향 통제 하에서만 정량**.
