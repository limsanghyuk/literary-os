# 인간 없는 정적-닻 자체평가 설계 (L4 v1.0, 2026-06-17)

**문서 ID**: LOS-STATIC-ANCHOR-SELFEVAL-L4-V1.0-2026-06-17 · **ADR 제안**: ADR-243
**전제**: 인간 작가 GT는 **최종 시험(Final Exam)으로 보류**(비용·시간). 그때까지 **정적 닻(대중·평단 검증 끝난 명작 원본)을 한계까지 짜내 자체 평가**로 LLM↔LLM 순환을 돌파한다.
**목표**: 인간 개입 없이 생성 모델 체급을 "명작 크래프트에 필적하는 모작" 수준까지 끌어올림. ※*독창성/예술성*은 정적 닻이 원리상 못 재므로 인간 최종시험에 남긴다(천장 명시).

## 0. 핵심 원칙
- 정적 닻으로 측정 가능 = "**기존 명작에 얼마나 가까운가**". 측정 불가 = "**새롭고 좋은가**"(=인간 최종시험).
- 정적 닻 자체평가의 도달 상한 = **뛰어난 모작(pastiche)**. 이는 결함이 아니라 *올바른 1단계 목표*. "모작 완성도 ≠ 예술성" 혼동 금지.

## 1. 세 메커니즘 (보강 포함)

### M1. Critic 영점 조준 — 열화판 판별 (Critic 자격 검정)
빈 `alignment_monitor` 닻을 인간 없이 *부분* 대체: 앵커를 "Critic↔인간GT"가 아니라 "명작 > 열화판 판별"로.
- **★등급화 열화 사다리**(쓰레기 아님): 미세→심각 N단계. 통과 = 단일 픽이 아니라 **단조 증가 변별곡선 + 올바른 순위**(변별 *해상도* 측정).
- **축별 표적 열화**: 복선 1개 제거 / 감정 평탄화 / 인과 단절 — 통과하려면 *그 축의 판단력* 필요(표면 흔적 탐지로 부정통과 차단).
- **자격 기준**: 사다리 전 구간에서 명작 우위 + 단조성 충족한 Critic만 자체평가 심판 자격. (필요조건; 신작 *생성* 판정 자격은 아님)

### M2. NextEpisodeBench — 은닉 정답 (보상의 주 닻)
명작 1~N화 정보(지식상태·잔향·긴장곡선) 주입 → N+1화 생성 → **실제 방영 N+1화(Hidden GT)와 쌍대 비교**.
- **★"닮음" 아닌 "필적"**: 실제 다음 화는 *유효한 한 갈래*. 측정은 "실제와 *동일 궤적*"이 아니라 "**크래프트 축에서 *필적*(쌍대 승/무)**". (단일정답 오류 차단)
- **암기 누출 가드**: 무명·비공개작 채점 + 생성↔실제 n-gram 중첩·base perplexity 임계 초과 작품 제외(유명작은 시연만).
- 측정 축: 감정 밀도·모티프 회수율(`drse` 잔향)·갈등 긴장도 — *유사도*가 아닌 *경쟁력*으로.

### M3. 분포 매칭 — ★음성 가드레일로만 (보상 목표 금지)
명작 통계(대사/지문 비, 비트별 감정어 빈도, 씬 길이)를 **상한·하한 가드**로만 사용.
- **금지**: 분포 일치를 *양성 보상*으로 → 평균회귀(mode-seeking, 밋밋한 중앙) + 굿하트(통계만 맞는 영혼 없는 텍스트). 명작은 평균 분포를 *일탈*한다 = 일치 보상은 탁월함을 억제.
- **허용**: *병리적 이상치*(대사 0% / 감정어 90% / 씬 길이 폭주)만 **감점·기각**. 비대칭 — "깨진 것을 거르되 전형성을 보상하지 않는다."

## 2. loop-C 배선 (기존 컴포넌트와 결합)
```
명작 corpus(scenes·drse·features) ── 정적 닻
   │
   ├─ M1 열화 사다리 ─► Critic 자격검정(alignment_monitor 앵커 대체) ─► 자격 통과 Critic만 심판
   │
   ├─ M2 NextEp(은닉 N+1화) ─► 생성 vs 실제 쌍대(필적) ─► 선호쌍(주 보상)
   │        ├ 위치스왑·무명작·누출가드(입력 정제)
   │        └ arbitration(공식 R ↔ Critic |z|>1.5σ 불일치 → 쌍 격리)  ← LLM↔LLM 순환 독립 차단
   │
   ├─ M3 분포 가드 ─► 병리 이상치만 보상 감점(전형성 무보상)
   │
   └─► DPO 학습(+KL 가드, 평균회귀·해킹 차단)
          └─► G_LOOPC_WINRATE 수용: ΔW(실명작 대비)>0 AND KL≤τ AND 구조게이트 비퇴행
                 ├ PASS → 어댑터 채택 → 라운드++
                 └ FAIL → 폐기·하이퍼/데이터 조정
```

## 3. 정직한 한계
- **천장 = 모작**: 세 메커니즘 모두 "명작 근접"만 측정 → 독창성은 인간 최종시험의 몫. (의도된 1단계 경계)
- **M1은 자격검정**: 열화 판별 통과 ≠ 신작 판정 타당. 부분 닻일 뿐.
- **순환 잔존**: arbitration(비-LLM 공식)이 *부분* 독립이나, 공식 자체가 작품성 τ≈0.2로 약함 → 완전 debiasing은 여전히 소수 인간 GT(최종시험)에서 닫힘.
- **소수쌍·단일라운드 노이즈**: 최소 N라운드 이동평균/표본 확대로만 신뢰.

## 4. 구현 우선순위 (개발자)
1. **M1 열화 사다리 + Critic 자격검정** — `alignment_monitor` 앵커를 명작>열화 사다리로 재배선(인간 없이 닻 채움). 가장 먼저, 자체평가 신뢰의 전제.
2. **M2 NextEpisodeBench** — 무명작 선정·누출가드·필적(쌍대) 채점·선호쌍 산출. 보상 주 닻.
3. **M3 분포 가드** — features 통계 상/하한, 병리 이상치 감점만(양성 보상 금지).
4. 기존 arbitration·KL·G_LOOPC_WINRATE에 위 신호 결합.

## 5. 한 줄
현 단계 자체평가 = "AI가 *맨땅에서 잘 썼나*"가 아니라 "**검증된 명작의 구조·리듬·전개를 얼마나 정교하게 리버스 엔지니어링해 *필적*하는가**". 정적 닻으로 모작 체급을 극한까지 올린 뒤, 마지막에 인간 메스로 예술성을 가른다. M3는 *가드레일*, M1은 *자격*, M2가 *주 닻*.
