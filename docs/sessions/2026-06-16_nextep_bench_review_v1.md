# 검토 보고서 — 연속-생성 평가(NextEpisodeBench) + 품질 판별 학습 (REVIEW v1.0)

**문서 ID**: LOS-NEXTEP-BENCH-REVIEW-V1.0-2026-06-16 · **단계**: ★검토(REVIEW) — 설계·승인 아님
**기준선**: HEAD `5a92d0a` V772 / v13.25.0 · **심사**: 최고 수석 아키텍트(CSA) × 최고 수석 컴파일러 전문가(CSC) 합의
**근거 가정(사용자)**: ① N화를 주고 AI가 N+1·N+2화를 *약간의 힌트*로 생성 → 원작 다음 화와 쌍대. ② 코퍼스는 걸작만 아니라 평작·졸작 혼재 → 좋은·나쁜 데이터 모두 학습하고 *품질을 판별*해야 한다.

## 0. 목적
사용자 가정을 **설계 이전 단계에서** 심사한다. 판정 대상은 두 관문(A 누출 오염 통제 / B 비평 평가 타당성) + 신규 차원(C 품질 층화·판별). "가능한가/멈춰야 하나"만 가린다.

## 1. 심사 구도
- **CSA**: 시스템·데이터 구조로 *구성에 의한 격리*가 되는가.
- **CSC**: 오염·품질·평가를 *측정·경계 지을 수 있는 양*으로 계측 가능한가.

## 2. 관문 A — 누출 오염 통제 (불가 시 중단)
- **쟁점**: 생성기(gpt-4o-mini/base)가 유명작(미생 등)을 사전학습에서 봤다면 "다음 화 생성"이 *회상*일 수 있어 품질이 허위 상승.
- **CSA**: held-out(A~Y 학습→Z 시험)로 우리 학습 누출은 막힌다. **CSC 반박**: base 사전학습 노출은 못 막고, 유명작은 표면 n-gram이 아닌 *플롯 비트의 구조적 기억*이라 surface 검사로 안 잡힌다.
- **합의(통제 3종)**: ① 무명·비공개작 채점(우리 코퍼스 다수가 비공개 입수작 → 노출 prior 낮음) ② 오염 계측 가드(생성↔실제 n-gram 중첩 + *실제 다음화의 base perplexity*가 비정상 저低 = 기억) → 임계 초과 작품 제외 ③ 자체 코퍼스 미세조정 생성기 + held-out.
- **판정**: ✅ **통제 가능(조건부)**. "0 누출"이 아니라 "학습 신호 왜곡 안 할 만큼 경계". **유명작은 시연만, 채점은 무명·비공개작.** 정지 조건 = 유명작을 채점에 고집 + 계측 생략. 그 한 경우만 피하면 **멈출 필요 없음**.

## 3. 관문 B — 비평 평가 타당성
- **CSC 핵심 구분**: ① *학습 신호용* = 완벽 심판 불요, 진짜 품질과 *양의 상관*만 충분하면 경사상승이 개선(실 DPO 증명에서 불완전 신호로도 지표 이동). → **지금 가능.** ② *"인간팀 필적" 주장용* = 인간 GT 보정 필수. → 보류.
- **단일 정답 오류 해소**: 실제 N화는 유효한 *한* 갈래일 뿐 → *유사도*로 채점하면 치명적이나, *크래프트 축 쌍대 품질 선호*로 채점하면 무력화. 게다가 "생성 N화 vs 실제 N화"는 같은 이야기의 **전편 대 전편(like-with-like)** → 심판 분산이 현행("생성 씬 vs 무관 명작 씬")보다 *오히려 작다*.
- **독립 앵커**: 공식 R은 LLM 아님 → 구조 축에서 LLM↔LLM 순환(작가-패널 80% 허위일치)을 깬다.
- **판정**: ✅ **학습 신호 수준 가능(현행보다 우수)**. natural-GT가 최대 confound(레퍼런스=비非베스트)를 제거하고 심판 신뢰도까지 올림. "필적 주장"은 인간 GT 후순위.

## 4. ★관문 C — 품질 층화: 좋은·나쁜 데이터 학습 + 판별 (사용자 신규)
코퍼스는 걸작·평작·졸작 혼재다. 이는 약점이 아니라 **자산**이나, 채널별로 다르게 다뤄야 한다.

### 4.1 "나쁜 데이터도 배운다"의 올바른 의미 — 모방 아닌 *대조 신호*
졸작을 *모방 목표*로 삼으면 안 된다. 졸작의 가치는 **DPO의 rejected(부정 예시)** 다 — 모델은 "하지 말 것"을 배운다. 즉 걸작=chosen, 졸작=rejected의 **대조쌍**이 자연 생성된다. 이는 연속-생성 쌍(생성<실제)과 **상보적**이다.

### 4.2 채널별 데이터 라우팅 (품질 라벨이 결정)
| 채널 | 무엇을 넣나 | 이유 |
|---|---|---|
| RAG(생성 정박) | **양질 편향**(걸작·수작) | 생성을 졸작 작법에 정박시키면 안 됨 |
| DPO/loop-C | **대조**(걸작 chosen / 졸작 rejected) + (실제 chosen / 생성 rejected) | 부정 예시가 학습 핵심 |
| Critic 보정 | **전 스펙트럼**(걸작~졸작) | 평가기가 품질을 가르는지 *검증*하려면 나쁜 것도 필요 |

### 4.3 품질 라벨 출처 (정직)
외부 권위 신호: **시청률·관객수·수상**(이미 meta_gt_drama 취합). ★주의: *인기 ≠ 작품성* — 흥행 졸작·무명 걸작 존재. 시청률은 인기, 수상은 다른 축. 따라서 품질 라벨은 **다신호 조합 + 노이즈 인정**이 전제. **모델 자기 critic으로 품질 라벨링은 순환 → 금지**(외부 신호로만).

### 4.4 ★Critic 판별 게이트 (가장 중요 — 선결 조건)
평가기(패널+공식)가 **알려진 걸작 > 졸작을 실제로 가르는가**를 *먼저* 증명해야, 연속-생성 판정을 신뢰할 수 있다.
- **현황(정직)**: FE-7 실측 = 공식 vs 드라마 명성 τ0.185(전체)·0.257(지상파)·0.267(수상). = **약하지만 양의 판별**. 패널은 더 나을 수 있으나 순환 위험.
- **함의**: 판별력이 *부분 충족*이다. → 연속-생성 학습 전, **걸작/졸작 라벨된 held-out에서 평가기 판별력(걸작>졸작 순위 일치율)을 측정·강화**하는 것이 선결. 판별 못 하면 학습 신호도 의심.
- **판정**: 🟡 **조건부 가능** — 외부 라벨로 평가기 판별을 먼저 검증·보정해야 함. 이게 통과해야 C가 학습에 기여.

## 5. 종합 판정 (검토 단계)

**결론: GO (검토 단계 승인, 설계 미착수).** 누출은 해결 불가 차단막이 아니라 *작품 선택+계측으로 경계 지어지는 양*이므로 가정은 멈출 사안이 아니다. natural-GT 연속 생성은 현 학습 패러다임 중 가장 타당하며, 품질 층화는 **졸작을 부정 예시로 쓰고 평가기 판별을 외부 라벨로 검증**할 때 학습을 강화한다. 단 **Critic 판별 게이트(걸작>졸작 실증)**가 선결 조건이다.
