# 외부 평가 기반 명작/평작/졸작 분류 — 1차 데모 (2026-06-16)

**목적**: 코퍼스 작품을 인터넷의 *전문가 평가 + 일반인 평가*로 조사해 **명작/평작/졸작** 외부 라벨을 산출. NextEpisodeBench 검토보고서의 **Critic 판별 게이트**(평가기가 걸작>졸작을 가르는지 검증) 입력. ※인기≠작품성 분리가 핵심.

## 방법론 (다신호 조합)
| 신호 | 출처 | 역할 | 주의 |
|---|---|---|---|
| 전문가 평점 | 씨네21 별점·네이버 평론가 | **작품성 1순위 축** | 표본 적을 수 있음 |
| 일반인 평점 | 네이버 관람객·씨네21 관객·시청자 | 대중 호응(작품성 2순위) | 팬덤 편향 |
| 흥행 | 관객수·시청률 | **인기 전용(작품성 아님)** | 흥행 졸작·무명 걸작 존재 |
| 수상·평론 합의 | 백상 등·평단 캐논화 | 작품성 권위 보정 | 시대 보정 필요 |

**등급 규칙**: 명작 = 전문가 高 + 캐논/수상 + 지속 호평. 평작 = 전문가 中 + 대중 호응(캐논 아님). 졸작 = 전문가 低 + 부정 합의. **흥행만 높고 평단 中이면 평작**(인기≠작품성).

## 핵심 관찰 (인기 vs 작품성 분리 실증)
- **태양의 후예**: 시청률 38.8%(역대급 인기) ≠ 평단 캐논. 흥행만으로 명작 라벨 금지 → **평작**. 이게 "인기≠작품성"의 대표 사례.
- **곡성**: 평론가 일부 혹평(김영진 "클라이맥스 최악")에도 캐논화 → **호불호와 작품성은 별개**. 단일 평점이 아닌 합의·지속성으로 판정.
- **미스터 소크라테스**: 흥행 성공(126만)이나 전문가 5.5·"산만" → 흥행≠품질의 역방향 사례. 졸작 경계.
- → 전문가 축을 1순위, 흥행을 인기 전용으로 분리하니 **인기작과 작품성작이 깔끔히 갈림**. 방법론 유효.

## 파이프라인 연결 (이 라벨의 용도)
1. **Critic 판별 게이트 검증**: 위 라벨을 정답으로, 우리 평가기(패널+공식)가 명작>평작>졸작 순위를 재현하는지 측정. 재현 못 하면 학습 신호 의심(현 FE-7 τ0.19~0.27 = 약한 양의 판별 → 이 라벨로 정밀 측정·강화).
2. **채널 라우팅**: RAG=명작 편향 / DPO=명작(chosen)·졸작(rejected) 대조쌍 / Critic 보정=전 등급.
3. ※모델 자기 critic으로 라벨링 금지(순환). 본 라벨은 **외부 신호 전용**.
