# ADR-199: G90 FL Gate — FL-1~FL-5 (V737)

**날짜**: 2026-05-29  
**상태**: ACCEPTED  
**서브페이즈**: SP-D.4

## Context

SP-D.4 D-M-10 Federated Learning PoC 완성 후 품질 검증 게이트가 필요하다.  
G90은 FL 서브시스템의 5개 핵심 축을 자동화 검사한다.

## Decision

`literary_system/gates/fl_gate.py`에 G90 FL Gate를 구현한다.

| 검사 ID | 설명 | 판정 기준 |
|---------|------|-----------|
| FL-1 | 최소 클라이언트 등록 | FLCoordinator.registered_clients ≥ 2 |
| FL-2 | FedAvg 가중 평균 정확성 | w_global = Σ(n_k/N)*w_k 오차 < 1e-9 |
| FL-3 | DP 프라이버시 예산 접근 | epsilon/delta/sigma 키 존재 + σ 수식 검증 |
| FL-4 | 수렴 감지 동작 | Δloss < threshold → converged=True |
| FL-5 | E2E 파이프라인 완료 | FLOrchestrator.run_federation() 정상 종료 |

`release_gate.py` GATES 리스트에 G90 엔트리 추가.

## Consequences

- FL-1~FL-5 5/5 PASS = G90 APPROVED
- SP-D.4 Phase D Exit Gate G95 전제 조건 충족
