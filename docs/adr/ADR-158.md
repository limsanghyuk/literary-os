# ADR-158: SP-D.2 개시 — MultiAgent Coordination Layer 아키텍처 설계

**날짜**: 2026-05-28
**버전**: v12.1.0 (V696)
**상태**: ACCEPTED (소급)
**결정자**: SP-D.2 Core Team

## Context

SP-D.2는 literary-os Phase D의 두 번째 서브 페이즈로, Literary OS에 완전한
MultiAgent Coordination Layer를 추가한다. Phase D 설계도(ADR-145)에 따라
에이전트 간 메시지 교환, 역량 등록, 협업 프로토콜, 충돌 해결, 워크플로우,
감독자 패턴, 로드 밸런서, 회로 차단기를 순차적으로 구현한다.

## Decision

`literary_system/agents/` 패키지 내 8개 핵심 모듈을 V696~V710에 걸쳐 구현:
- V696: sp-d.2 착수 기준선 및 에이전트 레이어 기초 설계
- V697: AgentMessage + AgentTask 스키마
- V698: CapabilityRegistry
- V699: CollaborationProtocol
- V700: ConflictResolver
- V701: TaskScheduler
- V702: AgentWorkflow
- V703: AgentSupervisor
- V704: agents/LoadBalancer
- V705: agents/CircuitBreaker
- V706: G84 Agent Coordination Gate
- V707: G85 Agent Workflow Gate
- V708~V709: SP-D.2 통합 테스트 + Exit Gate
- V710: SP-D.2 릴리즈

## Consequences

- MultiAgent Coordination Layer 완전 가동
- Gate G84/G85 검증 가능
- SP-D.3 Plugin + ZeroTrust 레이어 기반 마련
