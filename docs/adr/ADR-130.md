# ADR-130: Sudowrite 경쟁 흡수 아키텍처 (G72-2, SP-C.4)

**Date**: 2026-05-27  
**Status**: Accepted  
**Version**: V668 (v11.41.0)  
**Deciders**: Literary OS Architecture Board  

---

## Context

SP-C.4 경쟁 흡수 서브페이즈의 두 번째 타깃인 Sudowrite를 분석하고 핵심 기능 격차를
Literary OS 로드맵에 통합한다. C-M-11 의무 사항에 따라 IP 자문 커밋(IP-ADV-002)을
선행 수행한다.

**Sudowrite 개요**
- 영미권 소설가·단편 작가 대상 AI 글쓰기 보조 도구
- 구독 모델 ($10–$25/mo)
- 핵심 차별 기능: Wormhole 재작성, Story Bible, Describe 감각 확장, Canvas 플롯 보드

---

## Decision

### IP 자문 커밋 (IP-ADV-002)

| 항목 | 판정 |
|------|------|
| Wormhole 재작성 | LLM 프롬프팅 기법 — 독자 구현 가능 |
| Canvas 플롯 보드 | 공개 UX 관행 (Trello 류) — 위험 없음 |
| Describe 감각 확장 | 공개 창작 기법 — 독자 구현 가능 |
| Story Bible | Public-domain 템플릿 구조 — 위험 없음 |
| 실시간 Co-Pilot | 범용 스트리밍 API — Phase D 이관 |

### 기능 흡수 결정 (4 흡수 / 1 보류)

| 기능 | 결정 | Literary OS 구현 경로 |
|------|------|----------------------|
| StoryBible | 흡수 | `world.story_bible.StoryBibleAggregator` |
| WormholeRewrite | 흡수 | `SceneWeaver.rewrite_variations()` |
| DescribeSensoryExpansion | 흡수 | `EmotionEngine.SensoryExpander` |
| CanvasPlotVisualizer | 흡수 | `viz.plot_canvas.PlotCanvas` (Phase D) |
| RealtimeCoPilotSuggestion | 보류 | Phase D WebSocket 스트리밍 레이어 |

### G72-2 서브게이트 통과 조건

```
ip_advisory.cleared == True
len(absorbed_features) >= 3  # 실제: 4
len(rejected_features) <= 2  # 실제: 1
```

- **후속**: V669 Novelcrafter 흡수 (G72-3) 진행
