# ADR-239: Preflight 준수 복구 + 최종 무결성 감사 (V779)

## 상태
ACCEPTED (V779)

## 맥락
감사 결과: V767 train_local.py가 G32(literary_system/ 내 print 금지) 3건 위반 → Preflight FAIL 상태였으나, V767~V778 동안 `tail -1`로 로그저장줄만 확인하고 **판정(PASS/FAIL)을 미확인**. DEV_PROTOCOL_v3.0 RULE-0 준수 누락. 또 Preflight 로그가 docs/sessions(gitignore)라 허브 증거 부재.

## 결정
- train_local.py: CLI 콘솔 출력에 `_cli_demo` 마커(G32 정식 제외 규칙) 추가 → Preflight PASS 복구.
- 본 버전부터 Preflight 로그를 `git add -f`로 허브 적재(감사 증거 상설화).
- 최종 무결성 입증: G_INTEGRITY_MANIFEST 자기검증 PASS, G_CONNECTIVITY 0고립/86연결, Survival 54/54 ALIVE, V767~V778 249 TC PASS.

## 결과
Preflight PASS(exit 0, G32 0건). 8경고는 기존 gates/ 순환·고립(블록 아님, 신규 코드 무관). 로직/알고리즘 검증·무결성 입증 완료. 13.31→13.32.
## 대안
print→logging 전환: 광범위 변경(차후). 현재는 정식 제외 마커로 최소 수정.
