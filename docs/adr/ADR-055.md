## 결정

### Minimal-CLI v0.1 설계

```
literary analyze <scene_file>    — 5축 LOSConstitution 품질 분석
literary repair  <series_id>     — 시리즈 품질 진단 및 수리 제안
literary generate -e N -s M      — 장면 텍스트 생성 (LLM-0, 규칙 기반)
```

- **구현 위치**: `apps/cli/literary_cli.py` (click 기반)
- **LLM-0 원칙**: 외부 LLM 호출 0건, 모든 분석은 LOSConstitution/CorpusPipeline
- **출력 형식**: `--format text|json` 지원
