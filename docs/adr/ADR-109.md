# ADR-109: EditorAgent — 최종 통합·KoreanCadencePlanner 교정 (V649)

## 상태
승인됨

## 컨텍스트
최종 편집 단계에서 거부권 없이 초안을 통합해야 한다(C-M-09: editor_can_reject=False).
KoreanCadencePlanner가 있으면 한국 드라마 호흡을 보정한다.

## 결정
- `EditorAgent` + `EditedScene` 구현
- `finalize()`: 항상 산출물 반환, 거부 없음
- KoreanCadencePlanner 있으면 `.apply()` 호출; 없으면 기본 공백 교정
- CriticReport suggestions 반영 메모 생성
- LLM-0: 외부 API 직접 호출 없음

- 테스트: test_v649_editor_agent.py (30 TC)
