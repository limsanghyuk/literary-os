# ADR-044: GraphRealAdapter — networkx-optional 그래프 스토어

**상태:** Accepted  
**날짜:** 2026-05-20  
**버전:** V585 (9.0.0)  
**레이어:** L1 (DB 레이어)

## 컨텍스트

LOSDB Phase B 마지막 레이어. SQL(V582) + Vector(V584) REAL 구현 완료 후 Graph 레이어 REAL 구현 필요.
현재 `GraphMigrationAdapter`는 mock=True 상태로 실제 그래프 연산 없이 성공 시뮬레이션만 수행.

## 결정

1. **networkx-optional**: networkx 설치 시 가속, 미설치 시 순수 Python adjacency-dict fallback (V584 numpy-optional 패턴과 대칭)
2. **GraphRecord + GraphEdgeRecord**: 타입 안전 데이터클래스 + JSON 직렬화
3. **Migration.graph_ops**: `add_node | add_edge | remove_node | remove_edge` 연산 목록
4. **rollback 스냅샷**: `apply()` 전 `copy.deepcopy` 스냅샷 → `rollback()` 시 복원
5. **버전 9.0.0**: LOSDB Phase B 완료 기념 메이저 버전 범프

## 결과

- GraphRealAdapter(BaseMigrationAdapter) 구현 완료
- 43개 테스트 PASS
- Gate G44 (7체크포인트) PASS
- LOSDB Phase B 완전 완료: SQL + Vector + Graph 모두 REAL 구현

## LLM-0 원칙

외부 LLM 호출 없음. networkx는 순수 Python 그래프 알고리즘 라이브러리 (LLM 아님).
