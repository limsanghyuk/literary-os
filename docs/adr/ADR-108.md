# ADR-108: CriticAgent — 헌법 5축 평가 + NarrativeFitnessArbiter (V648)

## 상태
승인됨

## 컨텍스트
초안 품질을 측정하려면 헌법 5축(서사 일관성·감정 공명·캐릭터 일관성·페이싱·주제 깊이)과
NFA의 SELECT/MERGE/REJECT 결정을 결합해야 한다.

## 결정
- `CriticAgent` + `CriticReport` 구현
- PASS_THRESHOLD=0.65; 점수 ≥ 0.65 AND REJECT 아님 → passed=True
- Constitution 없으면 길이·키워드 기반 휴리스틱 사용
- `request_regeneration = not passed AND round_num < 3` (C-M-09)
- LLM-0: 외부 API 직접 호출 없음
