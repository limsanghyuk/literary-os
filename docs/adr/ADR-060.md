# ADR-060: Fine-tuning Pipeline Gate G54 (SP-B.1 완료 기준)

**상태**: Accepted
**날짜**: 2026-05-22
**버전**: V600 (v10.5.0)
**관련 ADR**: ADR-056 (LoRA Dataset), ADR-057 (Training Config), ADR-058 (Inference GW), ADR-059 (Eval 기준선)
**구현 파일**:
- `literary_system/gates/lora_finetuning_gate.py`
- `.github/workflows/finetune_ci.yml`

---

## 1. 맥락

V596~V599에서 SP-B.1의 모든 구성 요소가 구현되었다.
V600에서는 이 구성 요소들이 **수직 통합** 관점에서 E2E 동작하는지 검증하는
Gate G54가 필요하다.

Phase B 본안 보강 완료 요건:
- G53 + G54 PASS = SP-B.1 완료 선언 가능
- 테스트 6,382+ PASS (V595.2 기준 +200)
- LoRAArtifact 3-tag sha256 무결성
- 격주 학습 운영 CI 워크플로우 활성화

---

## 2. 결정

### 2.1 Gate G54 — 7체크포인트 수직 통합

| CP | 대상 | 검증 내용 | 실패 조건 |
|----|------|-----------|----------|
| G54-1 | PreTrainSafety | PII/Toxic/Copyright/Quality 4축 차단 | 차단 실패 1건이라도 |
| G54-2 | LoRADatasetBuilder | JSONL 산출 + sha256 content_hash 생성 | 샘플 0개 또는 hash 없음 |
| G54-3 | DatasetSplitter | 8:1:1 seed=42 재현성 | 분할 합계 불일치 또는 비재현 |
| G54-4 | ProvenanceLedger | sha256 체인 무결성 verify() | verify() False |
| G54-5 | LoRAJobRunner | dry_run() JobRunRecord 반환 | record None 또는 status 없음 |
| G54-6 | LoRAArtifact | 3-tag 생성 + tag_string 포맷 | seed/commit/dataset 누락 |
| G54-7 | FineTuneEvalPipeline | 5축 임계 PASS (stub 동일 텍스트) | any(axis.score < threshold) |
