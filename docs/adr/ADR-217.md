# ADR-217: AlignmentMonitor + G_LLM1_ALIGNMENT

## 상태
ACCEPTED (V757, Phase E.2)

## 맥락
LLM-1 Critic이 신뢰되려면 인간 작가 GT와 일치해야 한다(LW-5: LLM 심판 편향). 일치율을 게이트화한다.

## 결정
`critic/alignment_monitor.py`: Gold 쌍에서 critic 앙상블 판정 vs 인간 GT 다수결 일치율 측정. `G_LLM1_ALIGNMENT` ≥0.80(substantial). human_gt 재사용(절대 닻).

## 근거
일치율 미달 시 critic 프롬프트 개선/공식 recalibrate(Arbitration, V759). 인간 GT가 critic을 캘리브레이션.
## 결과
14 TC PASS, 픽스처 일치율 1.0. 실 작가 GT 도착 시 실측 대체.
## 대안 검토
일치율 측정 없이 critic 신뢰: 편향 방치(기각).
