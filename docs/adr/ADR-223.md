# ADR-223: PairwiseRewardModel (E.4 쌍대 보상)

## 상태
ACCEPTED (V763, Phase E.4)

## 맥락
loop-C 학습에 생성 품질 보상 신호가 필요. 절대점수 금지 원칙상 쌍대 승률로 보상한다.

## 결정
`learning/reward_model.py`: PairwiseRewardModel — critic 앙상블/주입 judge가 draft vs 실명작 풀 쌍대 판정 → 보상 R(x)=승률 ∈[0,1](tie=0.5). reward_from_pairs(누적 선호쌍 승률), ensemble_reward_model(3페르소나).

## 근거
DPO는 보상 불요(쌍대 직접). 본 보상모델은 RLHF/PPO·loop-C 격차 추적용. 보상↑=명작 근접.
## 결과
13 TC PASS. critic 앙상블 결선. GPU PPO/LoRA는 Phase F(finetune).
## 대안
절대 채점 보상모델: G_NO_ABSOLUTE_REWARD 위반(기각).
