# ADR-098 — LOSConstitution v2.0 Bayesian Weight Optimiser (SP-C.1, V631)

**날짜**: 2026-05-25
**상태**: Accepted
**작성자**: SP-C.1 설계팀
**기반**: ADR-054 (LOSConstitution v1.0, SP-A.7, V594)
**관련**: ADR-097 (Phase B Exit Gate G61, V630), ADR-081 (예약: G63 SelfLearningGate)

---

## 1. 맥락 (Context)

Phase B (V596~V630) 에서 LOSConstitution v1.0 (ADR-054) 은 **고정 가중치**
(drse=0.30, debt=0.20, arc=0.20, tension=0.15, prose=0.15) 를 사용했다.
이 값은 han-dramaturgy 전문가 3인의 사전 합의값으로, 드라마 장르 다양성과
작가별 스타일 편차를 충분히 반영하지 못할 수 있다.

Phase C SP-C.1 (V631~V645) 에서는 **Self-Learning Loop** 를 구축한다.
첫 번째 단계로 Bayesian Optimisation 을 도입하여 수집된 학습 샘플
`(씬 텍스트, 목표 점수)` 로부터 최적 가중치 w1~w5 를 **자동 탐색**한다.

---

## 2. 결정 (Decision)

`literary_system/constitution/los_constitution_v2.py` 에
`LOSConstitutionV2(LOSConstitution)` 을 구현한다.

### 2.1 Bayesian Optimisation (Optuna TPE)

- **라이브러리**: `optuna` (TPESampler, seed=42 기본)
- **탐색 공간**: 각 축 `wᵢ ∈ [0.02, 1.0]`, simplex 정규화 (`Σwᵢ = 1.0`)
- **목적 함수**: MSE(predicted_score, target_score) 최소화
- **기본 trial 수**: 100

### 2.2 Entropy 분포 제약 (C-M-05)

Shannon 엔트로피 H(w) = −Σ wᵢ · log₂(wᵢ) ≥ **1.5 bits**

| 설명 | 값 |
|------|----|
| 최솟값 (C-M-05 하한) | 1.5 bits |
| 5축 균등 분포 최댓값 | ≈ 2.322 bits |
| ADR-054 기본값 엔트로피 | ≈ 2.271 bits ✅ |

엔트로피 하한 1.5 bits 는 특정 축에 0.80+ 집중을 방지한다.
`entropy_constraint_pass()` 가 False 인 trial 은 즉시 pruned 처리된다.

### 2.3 영속화

`save(path)` / `load(path)` 로 최적화된 weights 를 JSON 으로 저장·복원한다.
파일 형식: `{ "version": "2.0", "adr": "ADR-098", "weights": {...}, ... }`

### 2.4 LLM-0 준수

외부 LLM API 호출 없음. 모든 스코어링은 V1 과 동일한 순수 텍스트 메트릭 사용.

---

## 3. 결과 (Consequences)

### 긍정적

- 드라마 장르·작가 스타일별 가중치 자동 맞춤
- 엔트로피 하한으로 과적합(single-axis 지배) 방지
- Optuna 미설치 환경에서도 V1 기본값 fallback 동작
- Phase C MetaLearner (V640~V644) 와 자연 연동

### 부정적·위험

- optuna 의존성 추가 (선택적 install)
- 학습 샘플 품질에 최적화 결과 의존
- n_trials=100 기준 단일 스레드 ~1~5초 (CPU 크기에 따라 다름)

---

## 4. 대안 검토

| 대안 | 기각 이유 |
|------|-----------|
| Grid Search | 5축 × 100단계 = 10⁵ trial, 비현실적 |
| Random Search | Bayesian 대비 수렴 속도 열세 |
| Gradient Descent | 비미분 텍스트 메트릭 — 직접 미분 불가 |
| 전문가 재조정 | 자동화 불가, SP-C.1 Self-Learning 목표 위반 |
