# ADR-165: AgentSupervisor — 에이전트 감독자 패턴 (V703)

**날짜**: 2026-05-28
**버전**: v12.1.7 (V703)
**상태**: ACCEPTED (소급)

## Context

에이전트 실패·에스컬레이션 발생 시 상위 감독자가 재시도·대체·종료
결정을 내려야 한다.

## Decision

`literary_system/agents/agent_supervisor.py`:
- `AgentSupervisor(strategy: SupervisionStrategy)`
- 전략 3종: ONE_FOR_ONE / ONE_FOR_ALL / REST_FOR_ONE
- `handle_failure(agent_id, reason) → SupervisionDecision`
- 최대 재시도 횟수 `max_restarts=3` 초과 시 에이전트 격리

## Consequences

- G85 체크포인트 E2(AgentSupervisor) 충족
- Erlang OTP 감독 트리 패턴 적용
