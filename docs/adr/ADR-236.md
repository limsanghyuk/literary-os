# ADR-236: 품질 라벨 자동 집계 엔진 (다신호 → 2축)

## 상태
ACCEPTED (V776, Phase E.4 확장)

## 맥락
V775 2축 라벨이 14편 수작업 데모. 전 코퍼스 확장을 위해 외부 신호(전문가·시청률·수상·채널)를 2축 점수로 자동 변환하는 엔진 필요. meta_gt_drama.py에 실 신호({시청률, expert 0~4, channel}) 존재.

## 결정
`quality/quality_aggregator.py`:
- commercial = 시청률/채널천장(케이블 천장 낮춤=채널 보정) 또는 관객수/1000만. craft = expert/4 + 수상보너스.
- **2축 독립 산출**(흥행이 craft에 안 샘 = 인기≠작품성). 외부 신호만(자기 critic 금지).
- from_drama_dict(meta_gt 포맷) → V775 classify로 등급. `tools/build_quality_labels.py` CLI.
- label_summary poor 이중계수 버그 수정.

## 결과
14 TC PASS. 실 meta_gt_drama 29편 자동 라벨링: 긍정 11(태양의후예=MASTERPIECE_BOTH, 흥행작 긍정 유지)·졸작 6·중립 12. 채널 보정으로 케이블 수작(미생 8.2%→comm 0.82) 정당 평가. 전 코퍼스 확장 기반. LLM-0(외부 신호 산술).
## 대안
1축 가중 합산: 인기≠작품성 분리 불가(기각). 2축 독립 산출 유지.
