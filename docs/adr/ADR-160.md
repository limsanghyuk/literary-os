# ADR-160: CapabilityRegistry — 에이전트 역량 등록소 (V698)

**날짜**: 2026-05-28
**버전**: v12.1.2 (V698)
**상태**: ACCEPTED (소급)

## Context

에이전트가 수행 가능한 역량(Capability)을 중앙 등록소에서 조회할 수 있어야
태스크 라우팅이 가능하다.

## Decision

`literary_system/agents/capability_registry.py`:
- `CapabilityRegistry` 싱글턴: `register(agent_id, capabilities: List[str])`
- `find_agents_by_capability(cap: str) → List[str]`
- `unregister(agent_id)` — 에이전트 종료 시 호출

## Consequences

- G84 체크포인트 E1(CapabilityRegistry 존재) 충족
- TaskScheduler가 역량 기반 에이전트 선택 가능
