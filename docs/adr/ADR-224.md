# ADR-224: RLAIF 오케스트레이션 브리지 (E.4 코어 닫기)

## 상태
ACCEPTED (V764, Phase E.4)

## 맥락
loop-C 선호쌍(V762)·보상모델(V763)을 실 생성기 학습(finetune LoRA)으로 연결해 "선호쌍→보상→학습" 사슬을 닫는다.

## 결정
`learning/rlaif_orchestrator.py` RLAIFOrchestrator: 선호쌍 → DPO 데이터셋 작성 + RLAIFTrainingSpec(base=Llama-3.1-8B, rank16, objective=dpo) + job_request(LoRAJobRunner 제출용). 최소 선호쌍 미달 시 blocked. **실 GPU 학습은 finetune.LoRAJobRunner(Phase F, GPU SLO 준수)** — 본 모듈은 LLM-0(준비만).

## 결과
13 TC PASS. 개발자 17쌍 → prepared(승률 0.588 baseline) 실증. learning→finetune 결선(G_CONNECTIVITY). **E.4 RLAIF 코어 완성**(실 LoRA 학습은 GPU/Phase F).
## 대안
sandbox GPU 학습: 불가·GPU SLO 위반(기각). 준비/트리거까지만.
