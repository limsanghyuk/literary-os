# ADR-231: 4070 첫 실 학습 실행 킷

## 상태
ACCEPTED (V771, Phase E.4 확장)

## 맥락
V762~V770은 전부 dry_run/PoC로, RLAIF 사슬이 실제 지표를 움직이는지 미검증. 첫 실 QLoRA 학습 1회로 메커니즘 증명 필요. 단 제 샌드박스는 GPU 없음 → 개발자 4070에서 실행할 패키지 제공.

## 결정
`learning/first_training_kit.py`: build_training_plan(데이터·VRAM·명령·경고), prepare_dpo(원본→DPO표준), make_smoke_dataset(저작권무관 합성), baseline_winrate, winrate_delta(전후 비교). 킷 `docs/sessions/2026-06-16_first_training_kit/`: README, SETUP_4070_WINDOWS, run_first_training.py(원샷), eval_winrate.py, sample_dpo_smoke.jsonl. **저작권: 실 텍스트 미동봉, 개발자 로컬 dpo_pairs.jsonl 지향.**

## 결과
12 TC PASS. 1회=메커니즘 증명(승률 이동 측정), 데이터<50쌍 경고. 개발자가 4070에서 스모크→실데이터 순으로 실행. LLM-0(준비·계획만). 다음=결과에 따라 데이터 확대+가드레일 반복.
## 대안
샌드박스 직접 학습: GPU 부재로 불가. 실행 킷 위임이 유일.
