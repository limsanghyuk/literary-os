# ADR-105: SelfLearningGate G63 — SP-C.1 완료 게이트 (V645)

## 상태
Accepted

## 날짜
2026-05-26

## 컨텍스트
SP-C.1(MetaLearnerCycle 4사이클 완주) 최종 수용 조건을 자동 검증하는 릴리즈 게이트가 필요하다.
V641~V644에서 MetaLearnerCycle 1~4 사이클이 완성되었으며, SP-C.1 완료를 선언하기 위한
수학적 합격 기준 3축이 ADR-098(V631)에서 정의되었다.

## 결정
G63 SelfLearningGate를 신설한다. 합격 조건:

1. **오염 0%** — ContaminationDetector가 학습 데이터셋에서 오염 검출 없음
   (`contamination_rate == 0.0`)
2. **KL < 0.05** — ConstitutionWeights 분포의 균등 분포 대비 KL 발산이 0.05 미만
   - `KL(w||uniform) = Σ w_i · log(w_i · n)`, n = 5 (헌법 축 수)
   - 표준 가중치 [0.30, 0.20, 0.20, 0.15, 0.15] → KL ≈ 0.03533 < 0.05
3. **α ≥ 0.70** — Krippendorff 알파(평가자 간 신뢰도) ≥ 0.70

## 구현
- `literary_system/gates/self_learning_gate.py`
  - `_kl_divergence_from_uniform(weights)` — KL 계산 함수
  - `SLGAxisResult` — 단일 평가 축 결과 데이터클래스 (axis_name, value, threshold, passed, detail)
  - `SelfLearningGateReport` — 게이트 평가 보고서 (report_id, evaluated_at, contamination_rate, kl_divergence, alpha, axes, passed, notes)
  - `SelfLearningGate` — 게이트 클래스 (evaluate / history / last_report / count / clear)
    - `store_path=":memory:"` 인메모리 모드 지원 (테스트 격리)
    - 기본 저장 경로: `data/losdb/self_learning_gate.jsonl` (JSONL 영속화)
  - `run_g63_gate()` — 7체크포인트 실행 함수 (release_gate.py 연동)
- `literary_system/gates/release_gate.py`에 GATES.append() 추가 → 62번째 게이트
- 테스트: `tests/unit/test_v645_self_learning_gate.py` (33 TC)
