# ADR-104: ConstitutionWeights 수렴 확인 설계 (V644)

- **상태**: Accepted
- **버전**: V644 (v11.15.0)
- **날짜**: 2026-05-26
- **담당**: Literary OS Core Team

---

## 컨텍스트

V631에서 LOSConstitutionV2는 Bayesian Optimisation으로 `ConstitutionWeights`(w1~w5)를 학습하고, C-M-05(ADR-098)로 `entropy(w) ≥ 1.5` 제약을 시행한다. 그러나 MetaLearnerCycle(V641~V643)은 MetaLearner outer-loop가 손실을 보정할 때 Constitution 가중치가 이 제약을 유지하는지 검증하지 않았다.

V644는 MetaLearnerCycle의 4번째(최종) 사이클로, MetaLearner 4사이클 완전체에서 Constitution 가중치 수렴을 검증하는 레이어를 추가한다.

---

## 결정

### 1. `WeightConvergenceReport` 값 객체

가중치 딕셔너리의 수렴 조건을 스냅샷으로 캡슐화하는 데이터클래스를 도입한다:
- `weights_dict`: 검사한 가중치 딕셔너리
- `weights_sum`: 합계 (정상: 1.0 근방)
- `entropy`: Shannon 엔트로피 (bits)
- `sum_ok`: `abs(sum - 1.0) ≤ WEIGHT_SUM_TOLERANCE(0.01)` 여부
- `entropy_ok`: `entropy ≥ WEIGHT_ENTROPY_MIN(1.5)` 여부 (C-M-05)
- `converged`: `sum_ok AND entropy_ok`

### 2. `MetaLearnerCycle.__init__`에 `constitution` 파라미터 추가

`constitution: Optional[LOSConstitutionV2] = None` 파라미터를 추가한다. 연결 시 각 사이클에서 `constitution._w.as_dict()`를 자동으로 수렴 검증한다. 미연결(`None`) 시 수렴 검증을 건너뛰고 `CycleReport.weight_convergence = None`.

### 3. `weight_convergence_check()` 퍼블릭 메서드

순수 함수 성격의 메서드로, 임의의 가중치 딕셔너리를 받아 `WeightConvergenceReport`를 반환한다. `run_cycle()` 내부 호출 외에도 독립적으로 사용 가능하다.

### 4. `CycleReport.weight_convergence` 필드 추가

각 사이클 리포트에 수렴 결과를 포함한다. `weight_converged` 프로퍼티는 `constitution=None`일 때 `True`로 간주하여 하위 호환성을 유지한다.

### 5. 상수

| 상수 | 값 | 근거 |
|------|----|------|
| `WEIGHT_SUM_TOLERANCE` | 0.01 | 부동소수점 누적 오차 허용 범위 |
| `WEIGHT_ENTROPY_MIN` | 1.5 | C-M-05 (ADR-098) 동일 임계값 |
| `WEIGHT_CONVERGENCE_MIN_CYCLES` | 2 | 이력 집계 최소 사이클 수 |

## 대안 검토

| 대안 | 거부 이유 |
|------|-----------|
| MetaLearner가 직접 Constitution에 접근 | MetaLearner는 nie/ 레이어로, constitution/ 의존성 추가 시 레이어 경계 위반 |
| 별도 ConvergenceMonitor 모듈 신설 | 단일 메서드 수준의 기능에 모듈 신설은 과도한 설계 |
| V645 Gate G63에서 일괄 검증 | MetaLearnerCycle 각 사이클에서 즉시 감지해야 빠른 피드백 가능 |

---

## 결과

- `weight_convergence_check()` 로 임의 가중치 수렴 즉시 확인 가능
- `weight_convergence_history()` 로 4사이클 이력 추적
- `CycleReport.weight_converged` 로 사이클 통과 조건에 수렴 여부 연계 가능
- V645 `SelfLearningGate G63`에서 4사이클 수렴 이력을 활용한 SP-C.1 완료 검증 가능
