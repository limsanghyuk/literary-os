# ADR-071: GenreTransferV2 + LoRAStackingAdapter — MultiWork v2 장르 전이 엔진 (V611)

- **상태**: ACCEPTED
- **날짜**: 2026-05-23
- **버전**: V611 (v10.16.0)
- **결정자**: Literary OS 수석 엔지니어

---

## 배경

SP-B.3은 MultiWork 협업 레이어를 구축하는 서브페이즈다. V607~V610을 통해
SharedCharacterDBV2, SharedWorldDBV2, MultiWorkOrchestratorV2, MultiWorkCIMV2가
완성되었다. V611에서는 이들 v2 컴포넌트와 연동하는 **장르 전이 엔진**과
**Multi-LoRA 스태킹 어댑터**를 도입한다.

기존 V565 `GenreTransferLearning`은 단순 선형 보간 전이만 지원하며,
CIM 보상 신호나 캐릭터/세계관 DB와 연동되지 않았다.

---

## 결정

### 1. `GenreTransferV2` (genre_transfer.py 확장)

`GenreTransferLearning`을 상속하여 MultiWork v2 생태계와 통합한다.

#### 보정 파이프라인 (`weighted_transfer`)

```
alpha_adjusted = clamp(alpha + boost_factor * CIM.reward_weighted_global_weight(project_id), 0, 1)
         ↓
transfer(source, target, alpha_adjusted)           # 기본 선형 전이
         ↓
+ char_reward_delta → emotional_intensity 보정      # 보상 평균 > 0.7 시 최대 +0.05
+ world_consistency_delta → description_density 보정 # 일관성 < 0.7 시 최대 +0.08
         ↓
coherence_score = same_target_genre_count / total_history
```

#### 장르 추천 (`recommend_genre`)

| CIM 보상 | 전략 | 근거 |
|---------|------|------|
| ≥ 0.5 | 가장 먼 장르 추천 (도전적 전이) | 보상이 높으면 독자 자극 가능 |
| < 0.5 | 가장 가까운 장르 추천 (안전한 전이) | 보상이 낮으면 점진적 조정 |

### 2. `LoRAStackingAdapter` (serving/lora_stacking_adapter.py)

Multi-LoRA 스태킹: 복수 LoRA 가중치를 선형 합산하여 단일 merged weight 생성.

```
merged[layer][key] = Σ_i (coeff_i × weight_i[layer][key])
조건: Σ coeff_i = 1.0 (허용 오차 1e-6)
```

#### 자동 계수 모드 (`genre_stack`)

CIM v2 `reward_weighted_global_weight`를 계수의 원시값으로 사용,
정규화 후 합산.

## 수학적 근거

### GenreTransferV2 보정 공식

**캐릭터 보상 → emotional_intensity**:
```
Δ_ei = min(0.05, max(0, (mean_reward - 0.7) × 0.25))
ei_new = min(1.0, ei_base + Δ_ei)
```
- 보상 평균 0.7 → Δ = 0, 보상 평균 0.9 → Δ = 0.05 (최대)

**세계관 일관성 → description_density**:
```
Δ_dd = min(0.08, (0.7 - consistency) × 0.4)  if consistency < 0.7 else 0
```
- 일관성 0.5 → Δ = 0.08 (최대), 일관성 0.7 이상 → 변화 없음

### LoRA 스태킹 합산 오차

부동소수점 누적 오차는 `round(..., 8)`로 제거.
계수 합 검증: `|Σ coeff_i - 1.0| < 1e-6`.

## 대안 검토

| 대안 | 기각 이유 |
|------|-----------|
| 비선형 전이 (sigmoid mixing) | 해석 가능성 저하, 과도한 복잡성 |
| LoRA SVD 합산 | 실제 텐서 없는 현 환경에서 구현 불가 |
| 장르 클러스터링 기반 추천 | 클러스터 수 결정 파라미터 추가 필요, V611 범위 초과 |
