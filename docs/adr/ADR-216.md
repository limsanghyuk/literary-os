# ADR-216: CriticEnsemble + G_LLM1_RAG (Pass7 패널 승격)

## 상태
ACCEPTED (V756, Phase E.2)

## 맥락
V754 5축 critic 도입 후, 이를 합의(앙상블)로 묶어 Pass7 패널을 승격해야 한다. 또 critic 호출의 RAG 의무를 게이트로 강제해야 한다(LLM-1 환각 억제).

## 결정
- `critic/ensemble.py`: CriticEnsemble — 5축 verdict→pairwise BT 합의(winner a/b/tie), `as_judge`로 Pass7 판정자 어댑터.
- `G_LLM1_RAG`: CriticContext/ensemble이 빈 rag_refs를 거부함을 검증.
- llm_critics 프롬프트 'WINNER:' 형식 강제 + `_parse` 폴백(V754 비교에서 tie 기본처리 문제 보강).

## 근거
- 합의=BT(절대점수 금지 유지). RAG 강제는 CriticContext 불변식 + 게이트 이중.
## 결과
- 69 TC PASS(critic 전체). ensemble은 Pass7 judge로 주입 가능 → 생성 본체와 결선.
## 대안 검토
- 단일 critic 판정: 편향·불안정(기각). 절대점수 합산: G_NO_ABSOLUTE_REWARD 위반(기각).
