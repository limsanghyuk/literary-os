# ADR-077: AutoPromotionGate G62 — SP-C.1 LoRA 자동 승격 게이트

## 상태
ACCEPTED

## 맥락
Phase C SP-C.1에서 MetaLearner(V631), WeightTracker(V632), PatternLibraryV2(V633),
RetrainingScheduler(V634)가 구현되었다.
이 파이프라인이 실행된 후 학습된 LoRA 모델을 "언제, 어떤 조건에서 PROMOTED 상태로 승격하는가"에
대한 명시적 기준이 없었다.
수동 승격은 오류 가능성이 높고 일관성이 없다.

## 결정
**AutoPromotionGate G62를 도입한다.**

조건:
1. **R(scene) ≥ 0.78**: LOSConstitution 기반 장면 품질 평균 점수
2. **롤백 횟수 = 0**: ConstitutionWeightTracker 기준 가중치 롤백 없음

두 조건을 모두 충족한 경우에만 자동 승격을 허용한다.

## 근거

### R_THRESHOLD = 0.78 선택
- Phase B에서 인간 평가자(3인)와 LOSConstitution이 합치한 최소 품질 기준
- 0.78 미만: 장면 구조·감정 호 중 하나 이상 미달
- 0.80 이상으로 설정하면 너무 엄격 — 초기 Phase C 모델의 현실적 달성 가능 수준 고려

### MAX_ROLLBACKS = 0
- 롤백이 한 번이라도 발생했다는 것은 가중치 불안정 증거
- 불안정한 가중치로 학습된 모델은 서빙에 부적합
- `force=True` 옵션으로 예외 처리 가능 (긴급 배포 시)

### LOSDB JSONL 영속화
- 승격 이력 감사 가능 (언제, 어떤 점수로 승격되었는가)
- V632 WeightTracker / V634 RetrainingScheduler 와 동일한 패턴

## 결과

### 긍정적
- 승격 기준 명문화 → 일관성 보장
- 승격 이력 JSONL로 감사 가능
- SP-C.1 Exit 조건(G62) 달성

### 부정적/제약
- 실제 R(scene) 측정은 EquivalenceTester(V591) 연동 필요 (V635에서는 골든셋 기반 검증)
- MAX_ROLLBACKS=0 기준이 지나치게 엄격할 수 있음 → V640+ 재검토
