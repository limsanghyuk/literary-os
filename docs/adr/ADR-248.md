# ADR-248: M3 분포 가드 실 corpus 재보정

## 상태
ACCEPTED (V787) — db/corpus_ko 실데이터 학습 반영. ADR-245(M3) 보정.

## 맥락
V784 M3 NORMAL_BANDS가 추정값. 개발자 db/corpus_ko(2,030작품/122,681씬) SceneFeature 전수 분석 결과 **dialogue_ratio 중앙값 0**(시나리오는 대사 0%도 정상)인데 기존 하한 0.10 → **실 명작 절반을 병리로 오기각**. (6지표 실험 M3 3/3 병리도 이 탓.)

## 결정
- NORMAL_BANDS dialogue_ratio 하한 0.10→0.0 (실 p02=0·p98=0.76).
- `CORPUS_FEATURE_BANDS`: 전수 122,681씬 p02/p98 — conflict[0,1.775]·energy[0,8.333]·motif[0.234,0.666]·curiosity[0.078,0.567]·dialogue[0,0.76]·n_chars[21,1742].
- `distribution_guard_features(features)`: 실 SceneFeature로 분포 가드(텍스트 추정보다 정확). 밴드 밖만 병리, 안=0(전형성 무보상).

## 결과
12 TC PASS. 대사 0% 시나리오 씬 더이상 병리 아님. 실 feature 가드 정상=0·병리 감점. 실 corpus 근거로 M3 정합. (백분위는 집계값, verbatim 비커밋). LLM-0.
## 대안
추정 밴드 유지: 실 명작 오기각(기각). 전수 백분위 채택.
