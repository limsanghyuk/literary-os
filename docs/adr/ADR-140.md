# ADR-140: V678 Enterprise 비용 제어 레이어 (EnterpriseCostControlGate G77)

## 상태
Accepted

## 날짜
2026-05-27

## 컨텍스트
SP-C.4 Enterprise Layer 4번째 안정화 단계. 멀티테넌트 환경에서 테넌트별
LLM 호출 비용을 추적하고 월간 예산을 집행하는 비용 제어 레이어가 필요하다.

## 결정
`literary_system/enterprise/cost_control.py` 신규 파일에 다음을 구현한다.
- `CostAlertLevel`: OK / WARNING(≥80%) / CRITICAL(≥95%) / EXCEEDED(≥100%)
- `CostCategory`: LLM_INFERENCE / EMBEDDING / FINETUNE / STORAGE / OTHER
- `EnterpriseCostBudget`: 테넌트별 월간 한도 + 임계값
- `CostEntry`: 단일 비용 항목 기록
- `EnterpriseCostAlert`: 임계값 초과 경보 (is_blocking은 EXCEEDED만)
- `EnterpriseCostReport`: 테넌트별 집계 (usage_pct, breakdown)
- `EnterpriseCostSuiteReport`: 전체 테넌트 집계
- `EnterpriseCostTracker`: 기록·조회·경보 생성
- `EnterpriseCostControlGate` (GATE_ID="G77"): 4-테넌트 데모 시나리오

게이트는 항상 PASS — 초과 시 경보만 발령하고 차단하지 않음.
이는 트래픽을 막지 않으면서 비용 가시성을 확보하는 정책이다.
