# ADR-134: 경쟁 흡수 증류 파이프라인 (DistillationExportPipeline, SP-C.4)

**날짜**: 2026-05-27
**상태**: 승인됨
**버전**: V672 (v11.45.0)
**관련 ADR**: ADR-129~133 (G72 시리즈)

## 컨텍스트

G72 5개 경쟁사 흡수 완료(ADR-129~133) 후, 추상적 "흡수 결정"을 Literary OS 내부 모듈 매핑으로 구체화하는 증류 단계가 필요하다.

## 결정

`DistillationExportPipeline` 구현:
- 22개 absorbed feature → 22개 `DistilledFeature` 명세 (DF-001~DF-022)
- 3단계 우선순위: IMMEDIATE (14) · NEXT (8) · DEFERRED (0)
- `export_roadmap()`: IMMEDIATE 우선 정렬된 실행 로드맵 반환
- G72-D 게이트: `export_ready=True` 및 `immediate≥10` 확인
