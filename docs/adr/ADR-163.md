# ADR-163: TaskScheduler — 멀티에이전트 태스크 스케줄러 (V701)

**날짜**: 2026-05-28
**버전**: v12.1.5 (V701)
**상태**: ACCEPTED (소급)

## Context

다수의 AgentTask를 우선순위·의존관계 기반으로 에이전트에게 배분하는
스케줄러가 필요하다.

## Decision

`literary_system/agents/task_scheduler.py`:
- `TaskScheduler(registry: CapabilityRegistry)` — 싱글턴
- `schedule(task: AgentTask) → agent_id` — 역량 매칭 + 가중 라운드-로빈
- `dequeue() → Optional[AgentTask]` — 우선순위 힙 기반

## Consequences

- G84 체크포인트 E2(TaskScheduler) 충족
- 에이전트 부하 균등화
