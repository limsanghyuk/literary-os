# ADR-132: NolanAI 경쟁 흡수 아키텍처 (G72-4, SP-C.4)

**Date**: 2026-05-27  
**Status**: Accepted  
**Version**: V670 (v11.43.0)

---

## Context

SP-C.4 G72-4: 영화·TV 대본 전문 AI 도구 NolanAI 분석. Literary OS 드라마 대본 생성 영역과 직접 겹치므로 대본 포맷·씬 헤딩·캐릭터 보이스 기능의 흡수 가치가 높다.

---

## Decision

### IP 자문 (IP-ADV-004)

- Final Draft (.fdx): 독점 포맷 → 오픈 Fountain 포맷으로 대체
- 나머지 기능 전체: 공개 창작 방법론 / 범용 기술 → 위험 없음

### 기능 흡수 (5흡수 / 1보류)

| 기능 | 결정 | Literary OS 경로 |
|------|------|-----------------|
| ScriptFormatEngine | 흡수 | `prose.script_formatter.ScriptFormatEngine` (KoreanDrama 포맷) |
| SceneHeadingAutocomplete | 흡수 | `FractalPlotTree.SceneNode.structured_heading` |
| CharacterVoiceConsistency | 흡수 | `ArcConsistencyChecker.CharacterVoiceChecker` |
| BeatBoardVisualization | 흡수 | `PlotCanvas.script_beat` (Phase D) |
| ProductionBreakdown | 흡수 | `production.breakdown.ProductionBreakdown` |
| FinalDraftExport | 보류 | Fountain 오픈 포맷 대체 (IP 리스크 중간) |

---

## Consequences

- G72-4 통과; 한국 드라마 대본 포맷·씬 헤딩·캐릭터 보이스 흡수 경로 확정
- Final Draft 대신 Fountain 포맷 지원 — ManuscriptExporter 확장 예정
- **후속**: V671 Jenova 통합 흡수 + G72-5 + G72 통합 게이트
