# ADR-218: CorpusGate(G_LLM1_SAFETY) + LLM1Metrics(G_LLM1_COST)

## 상태
ACCEPTED (V758, Phase E.2)

## 맥락
LLM-1 안전 운영: 코퍼스가 빈약하면 critic 신뢰 불가 → 차단. 외부 LLM 비용 폭주 방지.

## 결정
- `critic/corpus_gate.py` G_LLM1_SAFETY: 코퍼스<50편 시 critic 차단.
- `critic/llm1_metrics.py` G_LLM1_COST: 월 ≤$50 hard/$30 soft, 모델별 가격 추적(gpt-4o-mini/4o/5/5-mini/5-chat).

## 결과
16 TC PASS. 5축 측정의 '비용'·'안전성' 원천. 현 코퍼스 205~395편 → SAFETY 통과.
## 대안
무제한 critic: 환각·비용 위험(기각).
