# ADR-138: Enterprise 성능 벤치마크 레이어 (SP-C.4 안정화 2)

## Status
Accepted

## Date
2026-05-27

## Context
V673(SLO)·V674(Revenue) 구현 후 성능 특성이 명시적으로 측정·검증되지 않았다.
프로덕션 배포 전 SLO 모니터링 및 Revenue 계산 레이어가 임계값 내에서 동작함을
자동화된 벤치마크 게이트로 보장할 필요가 있다.

## Decision
`literary_system/enterprise/benchmark.py` 신설:
- `BenchmarkRunner`: 샘플 목록 → avg/p50/p99/throughput 통계 계산
- `BenchmarkGate` (GATE_ID="G75-BM"): SLO_MONITOR/REVENUE_CALCULATOR/FULL_PIPELINE 3종 데모 벤치마크
- 임계값: SLO avg≤50ms/p99≤200ms/tput≥20rps, Revenue avg≤30ms/p99≤120ms/tput≥30rps
- `time_call()`: 실제 함수 호출 시간 측정 유틸리티

## Consequences
- Enterprise 레이어 성능 회귀 자동 감지 가능
- Release Gate 76/76 PASS 유지
- 테스트 총계: 8,708 TC
