# ADR-193: G86 API Completeness Gate 신설 (DEFECT-2 수정)

**날짜**: 2026-05-29
**상태**: accepted
**범위**: literary_system/gates/api_completeness_gate.py, SP-D.2 DEFECT-2

## 배경

SP-D.2 mega-commit(V696~V715) 중 G86 게이트 파일이 생성되지 않아 DEFECT-2로 기록되었다.
V731에서 소급 신설한다.

## 결정

studio_api 앱 레이어 6축(A1~A6) 검증 게이트 신설:
- A1: OpenAPI 스키마 — 라우터 6종 이상 등록
- A2: 엔드포인트 응답 코드 완결성 — 핵심 5개 라우터 파일 존재
- A3: 인증 헤더 표준 — Bearer 토큰 미들웨어
- A4: 페이지네이션 패턴 — limit/offset 파라미터
- A5: 에러 응답 포맷 표준 — HTTPException/detail 필드
- A6: Rate-limit 헤더 — X-RateLimit-*/RateLimitBucket

## 결과

GATES 90 → 91. G85 직후, G87 직전 삽입. 6/6 PASS 검증 완료.
