# ADR-249: SP-E.10 졸업 (V795 Phase E Exit) — per-token loop-C 5/5 ADOPT → v14.0.0

## 상태
ACCEPTED (V795, 2026-06-24) — 집 RTX 4070 실측. graduation_invariant 6/6 충족(violations 0).

## 맥락
SP-E.10 졸업 조건 = 누적 어댑터 루프 **말미 5라운드 연속 adopt**(+ Σ쌍수≥250 · 전 라운드 W₁ CI하한>0.5 · length_rule_rate≤0.60 · c3 PASS). 선행 시도 실패 이력:
- v1(ab3735c): 누적 base-KL 폭발 → R2~5 전부 rollback (0/5).
- v2(2026-06-22): 혼합신호 **포화**(R2에 W=1.0) → mastery-maintain 도입했으나 maintain이 연속 adopt 스트릭을 끊어 졸업 구조적 불가 (graduated=False, R4 rollback).
- ★실코드 확인(loopc_closure.py): `graduation_invariant`은 말미 연속 adopt만 세고 maintain=rollback과 동일하게 스트릭을 끊음. CumulativeLoop는 maintain 자체가 없음(adopt/rollback 2지선다). → 졸업엔 **라운드마다 더 어려운 신호로 5연속 진짜 adopt**가 필수.

## 결정
**Path B "하드 신호 커리큘럼"** 채택(`tools/loop_c_4070_kit/`):
- chosen=show(감정어 금지) / rejected="능숙하나 평면적" tell. 격차를 좁혀 base W≈0.58로 묶어 포화 회피.
- 커리큘럼: 라운드 오를수록 rejected 완성도↑(level R1:2→R5:5), held는 level3 고정 → 매 라운드 "조금씩만" 이기게 = 5연속 진짜 학습 유도.
- 트레이너 `train_4070_cumulative_v3.py`: maintain 제거(adopt/rollback만), length_rule_rate 실측, rollback 시 진입 전 어댑터 복원. 단일 held=250.

## 정직한 경계 (조기 성공 선언 금지)
- 졸업 신호는 **show/tell 한 축**(craft 한 칸). "작가팀 대체" 거시 기획(16/24부작)은 여전히 빈칸 = 차기 단계(LLM-2 거시플래너, [[project-llm2-direction-seed]]).
- rejected=AI 생성 평이체. 인간 명작 직접대조는 최종시험으로 남음.
- 3-전문가팀 심의 경고: floor는 방향(측정 쉬운 축 쏠림)까진 못 막음 — 차기 단계서 양성축·축간 균형 메타게이트 필요.
