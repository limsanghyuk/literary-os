# ADR-214: LLM-1 Critic 진입 — CriticInterface 5축 + G_LLM1_BOUNDARY

## 상태
ACCEPTED (V753, Phase E.2 진입)

## 맥락
생성 본체 Pass1~7 배선(V752)·Pass5 실 LLM E2E 입증 후, 외부 LLM을 평가 보조(Critic)로 정식화하는 LLM-0→1 전이 진입. 단 corpus/·constitution/·finetune/ 의 LLM-0는 보존해야 한다.

## 결정
1. `literary_system/critic/` 신설 — 외부 LLM 허용 레이어(평가 보조). CriticInterface(추상)+5축(D-E2-4 확정: STRUCTURE/CHARACTER/DIALOGUE/EMOTION/GENRE)+CriticVerdict(쌍대 winner a/b/tie, 절대점수 금지).
2. `G_LLM1_BOUNDARY` 게이트 — corpus/constitution/finetune 에 외부 LLM 호출/임포트 정적 0건 강제.
3. critic 합의는 validation.pairwise.bt_scores로 집계(절대점수 금지 유지, G_CONNECTIVITY 결선).

## 근거
- 번호 정합: ADR-211~213 사용됨 → E.2 ADR는 214부터(본 ADR).
- 기존 자산 결선: base.aggregate_verdicts→pairwise BT, 후속 critic_ensemble←Pass7 패널, alignment←human_gt.

## 결과
- 긍정: LLM-1 경계가 정적으로 강제됨. base는 추상(실 LLM은 V754~ 서브클래스). 27 TC PASS.
- 유의: 실 LLM critic 비용·일치율은 V754~V759에서 G_LLM1_COST/ALIGNMENT로 관리.

## 대안 검토
- critic을 절대점수 채점기로: G_NO_ABSOLUTE_REWARD 위반(기각). corpus/에 LLM 허용: LLM-0 붕괴(기각).
