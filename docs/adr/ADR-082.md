# ADR-082: FeedbackIntegrator — 인간 피드백 통합기 (V640)

## 상태
Accepted (2026-05-26)

## 컨텍스트
SP-C.1 Self-Learning Loop의 10번째 컴포넌트.
DataAugmentationController(V639)가 데이터 증강을 담당한다면,
FeedbackIntegrator(V640)는 인간 검증자의 피드백을 수집·집계하여
가중치 보정 신호를 생성한다.

ConstitutionEvalV2(V637)의 자동 평가와 인간 검증을 연결하는 핵심 인터페이스.

## 결정

### 피드백 유형 4종

| 유형 코드 | 설명 |
|---|---|
| SCORE_CORRECTION | 자동 평가 점수 보정 (human override) |
| LABEL_REVISION | 레이블 수정 (오분류 정정) |
| STYLE_ANNOTATION | 문체 주석 (드라마 도메인 특화) |
| REJECTION | 샘플 거부 (품질 기준 미달) |

### 신호 생성 로직
- `has_signal = feedback_count >= MIN_FEEDBACK_FOR_SIGNAL(3)`
- `signal_strength = |avg_delta| * CORRECTION_WEIGHT(0.8) + rejection_rate * REJECTION_PENALTY(0.5)`

### 핵심 클래스
- `FeedbackRecord`: 단일 피드백 (correction_delta() 메서드 포함)
- `IntegrationResult`: 집계 결과 + 신호 강도
- `FeedbackIntegrator`: 수집·집계 엔진

### 영속화
- LOSDB JSONL append-only (`:memory:` 모드 지원)
