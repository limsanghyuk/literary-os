# ADR-195: FedAvg 집계 알고리즘 스텁 (V733)

**날짜**: 2026-05-29
**버전**: v12.5.2 (V733)
**상태**: ACCEPTED
**결정자**: SP-D.4 FL 팀

## Context

V732에서 FLCoordinator 골격이 완성되었다. 다음 단계로 Federated Averaging
집계 알고리즘을 구현해야 한다. V733에서 FedAvg 스텁을, V734~V735에서 FLClient
시뮬레이터와 차분 프라이버시 노이즈를 완전 구현한다.

## Decision

`literary_system/federation/fedavg.py`:
- `FedAvgAggregator(dp_noise_scale=0.0)` — V733 스텁
- `aggregate(client_states, round_num) → FLGlobalModel`
  - FedAvg 공식: `w_global = Σ (n_k / N) * w_k`
  - 샘플 수 가중 글로벌 손실 계산
  - `dp_noise_scale` 파라미터 시그니처 확보 (V734에서 실제 DP 노이즈 적용)
- `history: List[FLGlobalModel]` — 라운드별 집계 기록
- `loss_trend() → List[float]` / `is_improving() → bool` — 수렴 모니터링

## Rationale

FedAvg(McMahan et al., 2017)는 연합 학습의 표준 집계 알고리즘이다.
가중 평균 방식은 데이터 이질성(Non-IID)에도 수렴을 보장한다.

## Consequences

- V734: FLClient 로컬 훈련 시뮬레이터 + FLPrivacyNoise DP
- V735: FedAvg + FLClient 통합 테스트 (≥30 TC)
- G90 FL Gate FL-1~FL-5 검증 가능
