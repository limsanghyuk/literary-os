# ADR-166: agents/LoadBalancer — 에이전트 계층 로드 밸런서 (V704)

**날짜**: 2026-05-28
**버전**: v12.1.8 (V704)
**상태**: ACCEPTED (소급)

## Context

TaskScheduler가 배분한 태스크가 특정 에이전트에 편중되지 않도록
런타임 부하 지표 기반 재분배가 필요하다.

## Decision

`literary_system/agents/load_balancer.py`:
- `AgentLoadBalancer(agents: List[str])` — 에이전트 ID 목록
- 알고리즘: LEAST_CONNECTIONS 기본, WRR 옵션
- `select(task: AgentTask) → agent_id`
- `report_load(agent_id, load: float)` — 에이전트 자기 보고

## Consequences

- G85 체크포인트 E3(LoadBalancer) 충족
