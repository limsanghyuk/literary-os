# ADR-118: B2B Partner API — OAuth 2.1 + Webhook + 1,000 RPM

## 상태
Accepted (V658, 2026-05-27)

## 맥락
PublicSDK(V656)와 OpenAPI 문서(V657)를 갖춘 후, B2B 파트너에게
안전하고 확장 가능한 API 접근 권한을 부여해야 한다.

## 결정

### 인증: OAuth 2.1 Client Credentials
- `OAuth21Manager`: 파트너 등록 → client_id/secret 발급 → AccessToken 발급 → 검증
- 토큰 TTL: 3,600초 (기본), 환경별 조정 가능
- 시크릿: SHA-256 해시만 저장 (평문 보관 금지)
- 스코프: `analyze` / `repair` / `predict` / `generate` 4종

### Webhook: HMAC-SHA256 서명
- 이벤트 유형: `scene.analyzed` / `scene.generated` / `scene.repaired` /
  `feedback.received` / `quota.warning` / `quota.exceeded`
- 서명 헤더: `X-LOS-Signature: sha256=<hex>`
- 파트너별 독립 엔드포인트 등록/해제

### RPM 제한: 슬라이딩 윈도우 60초
- 기본값: 1,000 RPM
- 파트너별 독립 카운터
- 초과 시: `PartnerQuotaError` (HTTP 429 매핑)

### B2BPartnerAPI
3개 레이어를 통합하는 퍼사드:
```
B2BPartnerAPI
  ├── OAuth21Manager    — 인증
  ├── WebhookManager    — 이벤트
  ├── _PartnerRateLimiter — RPM
  └── LiteraryOSClient  — 코어 SDK
```
