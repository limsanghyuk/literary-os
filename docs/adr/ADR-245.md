# ADR-245: 정적-닻 자체평가 M3 — 분포 음성 가드레일

## 상태
ACCEPTED (V784) — ADR-243 설계의 M3 구현. M1(자격)·M2(보상 닻) 위.

## 맥락
명작 통계로 생성물 평가 시 분포 일치를 양성 보상하면 평균회귀(밋밋한 중앙)+굿하트(통계만 맞는 영혼없는 텍스트). 명작은 평균을 *일탈*하므로 일치 보상은 탁월함을 억제.

## 결정
`critic/distribution_guard.py`:
- compute_stats: 대사비율·감정어비율·평균문장길이·씬길이.
- distribution_guard: 명작 정상밴드 *밖*(병리)만 감점(PATHOLOGY_PENALTY -1/건), 밴드 안=0(**보너스 없음**). 누적≤기각임계 → 기각.
- apply_guard_to_reward: reward+penalty(≤0), 기각시 강한 하한. **비대칭**: 깨진 것 거르되 전형성 무보상.
- 한국어 단문체 고려(avg_sentence_len 하한 2.0).

## 결과
13 TC PASS. 정상 씬 penalty=0(무보상)·대사0%·감정어폭주만 감점/기각. 평균회귀·굿하트 차단. LLM-0(순수 통계). **자체평가 3종(M1 자격·M2 보상닻·M3 음성가드) 완성**. 다음=loop-C에 M1/M2/M3 통합 배선.
## 대안
분포일치 양성보상: 평균회귀·굿하트(기각). 음성 가드만 채택.
