# ADR-244: 정적-닻 자체평가 M2 — NextEpisodeBench (은닉 GT 쌍대)

## 상태
ACCEPTED (V783) — ADR-243 설계의 M2 구현. 선결 M1(V782 critic 자격검정) 통과 후 진입.

## 맥락
M1로 critic 자격을 세웠으니, 보상의 *주 닻*인 NextEpisodeBench 진입. 명작 1~N화 → N+1화 생성 → 실제 방영분(은닉 GT)과 쌍대. 핵심=닮음이 아닌 **필적**(실제 다음화는 유효한 한 갈래).

## 결정
`critic/next_episode_bench.py`:
- `run_next_episode_bench(items, generate, judge, critic_qualified)`: context→생성→실제와 쌍대(위치 스왑 무작위)→필적률/순승률→loop-C 선호쌍.
- **누출 가드**: 유명작 채점 제외(시연만) + 생성↔실제 n-gram 중첩>0.25 제외(암기 의심).
- **M1 선결 강제**: critic_qualified=False면 채점 거부.
- 결과 pairs → learning.loop_c PreferencePair(실제=chosen·생성=rejected) = 보상 주 신호.

## 결과
14 TC PASS. 데모: 평이 생성→필적률 0(실제 우위·학습여지 큼), 유명작 제외, 자격미달 critic 채점거부. 위치스왑 결정성(seed). LLM-0(가드 결정론, judge/generate만 LLM-1). 다음=M3 분포 음성가드.
**천장**: 필적(모작 근접)까지. 독창성은 인간 최종시험.
## 대안
유사도 채점: 단일정답 오류(기각). 크래프트 축 쌍대 필적으로.
