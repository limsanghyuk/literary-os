# ADR-133: Jenova 경쟁 흡수 (SP-C.4, G72-5, G72 통합)

**날짜**: 2026-05-27
**상태**: 승인됨 (Accepted)
**버전**: V671 (v11.44.0)
**관련 ADR**: ADR-129 (G72 게이트 프레임워크), ADR-130~132 (G72-1~4)

---

## 컨텍스트

SP-C.4 경쟁 흡수 서브페이즈 마지막 대상: **Jenova** (한국 드라마 특화 AI 서사 플랫폼).

NovelAI(G72-1)·Sudowrite(G72-2)·Novelcrafter(G72-3)·NolanAI(G72-4) 흡수 완료 후,
Jenova의 한국 드라마 특화 기능(장르 혼합·감정 피크·서사 일관성·인물 관계망)을 분석하여
Literary OS에 적합한 방식으로 흡수한다.

G72-5 서브게이트 통과 후 **G72 통합 게이트**를 신설하여 5개 경쟁사 전체 흡수 완료를 선언한다.

---

## 결정

### 흡수 기능 (4종)
| 기능 | 매핑 전략 |
|------|----------|
| KoreanGenreBlending | GenreTransferEngine 확장 — 공개 장르 이론 기반 |
| EmotionalPeakScheduler | NarrativeTensionCurve 위에 피크 타이밍 레이어 추가 |
| NarrativeCoherenceValidator | ArcConsistencyChecker 확장 — 멀티 아크 검증 강화 |
| CharacterRelationshipMapper | SharedCharacterDB v2 관계 레이어 추가 |

### 거부/연기 기능 (2종)
| 기능 | 사유 |
|------|------|
| PredictiveAudienceFeedback | IP 리스크 HIGH (독점 ML 모델) → ReaderFeedback 루프로 대체 |
| RealTimeCollab | 협업 프로토콜 특허 위험, 인프라 복잡도 높음 → Phase D 계획 |

### IP 자문 커밋
- **IP-ADV-005** `cleared=True` — 6개 항목 검토 완료

### Gate
- **G72-5**: Jenova 단독 서브게이트
- **G72**: 5개 경쟁사(NovelAI·Sudowrite·Novelcrafter·NolanAI·Jenova) 통합 게이트 — all_passed=True
