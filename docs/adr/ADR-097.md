# ADR-097: G61 Phase B Exit Gate 7축 강화 (V630)

**상태**: Accepted  
**날짜**: 2026-05-25  
**결정자**: Literary OS 아키텍처 위원회  
**Supersedes**: ADR-080 (G61 Phase B Exit Gate 6축 판정)

---

## 컨텍스트

V620에서 ADR-080을 통해 G61 Phase B Exit Gate를 C1~C6 6축으로 정의하였다.  
V621~V629(Phase B 완료 단계)에서 SP-B.2 인터페이스 트레이스 요구사항 5건(P-IF-01~05)을 구현하였으나, Exit Gate가 이를 명시적으로 검증하지 않았다.

V630은 Phase B 완전 종료를 선언하는 마지막 버전이다. 다음 두 가지 강화가 필요하다:

1. **C7 InterfaceTrace**: P_IF_TRACE_REQUIRED 5건 동적 임포트 검증을 Exit Gate에 통합하여, 인터페이스 존재 및 핵심 속성이 런타임에 보증됨을 Gate 수준에서 확인한다.

2. **MIN_TESTS 상향 (6700 → 7000)**: Phase B 전 단계(V621~V630)에서 60 TC씩 추가한 결과 누적 테스트 수가 7000을 초과하였다. 운영 신뢰성 기준을 이 수치로 격상한다.

---

## 결정

### C7: verify_interfaces_trace()

`phase_b_exit_gate.py`에 `verify_interfaces_trace()` 함수를 신설하고,  
`P_IF_TRACE_REQUIRED` 상수에 다음 5건을 정의한다:

| ID     | 모듈                                         | 클래스/함수              | 검증 속성                     |
|--------|----------------------------------------------|--------------------------|-------------------------------|
| P-IF-01 | literary_system.llm_bridge.agent_envelope    | AgentEnvelope            | agent_id, role                |
| P-IF-02 | literary_system.llm_bridge.agent_envelope    | AgentRoutingPolicy       | decide_for_agent              |
| P-IF-03 | literary_system.multiwork.reader_feedback_ingest | ReaderFeedback       | comment, engagement_seconds   |
| P-IF-04 | literary_system.serving.model_serving_endpoint | get_api_version_response | (callable 존재)              |
| P-IF-05 | literary_system.audit.atia_metadata_auditor  | ATIAMetadataAuditor      | export_package                |

`run_phase_b_exit_gate()`에 `_if_trace_override` 파라미터를 추가하여 테스트 주입이 가능하도록 한다.

### MIN_TESTS: 6700 → 7000

```python
MIN_TESTS: int = 7000  # V630 기준 (V620: 6700)
```

C6 체크포인트는 7000 미만 시 FAIL로 판정한다.

---

## 결과

- G61 체크포인트 수: 6 → **7** (C1~C7)
- Phase B Exit 기준: 7축 모두 PASS + MIN_GATES=60 + MIN_TESTS=7000
- `run_g61_gate()` gate_name 문자열: "SP-B 6축" → "SP-B 7축 완료 판정 (V630)"
- 테스트: `test_v630_phase_b_exit_v2.py` 60 TC 60/60 PASS
