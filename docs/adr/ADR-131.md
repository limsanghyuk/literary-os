# ADR-131: Novelcrafter 경쟁 흡수 아키텍처 (G72-3, SP-C.4)

**Date**: 2026-05-27  
**Status**: Accepted  
**Version**: V669 (v11.42.0)

---

## Context

SP-C.4 G72-3: 장편 소설 전문 플랫폼 Novelcrafter 분석 및 Literary OS 로드맵 통합.
IP 자문 커밋 IP-ADV-003 선행 수행 (C-M-11 의무).

---

## Decision

### IP 자문 (IP-ADV-003) — 전체 클리어

| 기능 | IP 판정 |
|------|---------|
| Codex WorldDB | 공개 DB 패턴 — 독자 구현 가능 |
| Scene Outline | 범용 워드프로세서 기능 — 특허 없음 |
| Chapter Beat Sheet | 공개 창작 방법론 — 위험 없음 |
| AI Draft Generate | LLM 범용 기술 — 독자 구현 |
| Series Management | 공개 UX 패턴 — 위험 없음 |

### 기능 흡수 결정 (5 흡수 / 1 보류)

| 기능 | 결정 | Literary OS 경로 |
|------|------|-----------------|
| CodexWorldDB | 흡수 | `SharedWorldDB.EntityTree` |
| SceneLevelOutline | 흡수 | `FractalPlotTree.SceneOutlineEditor` |
| ChapterBeatSheet | 흡수 | `EpisodeStructureCalculator.ChapterBeatGenerator` |
| AIDraftGenerate | 흡수 | `SceneGenerationPipeline.OutlineTrigger` |
| SeriesManagement | 흡수 | `MultiWorkCore.SeriesGrouping` |
| OfflineLocalStorage | 보류 | Phase D 로컬 에디터 클라이언트 |

### G72-3 서브게이트 통과 기준

```
ip_cleared == True
absorbed ≥ 3  # 실제: 5
rejected ≤ 2  # 실제: 1
```

---

## Consequences

- G72-3 통과, 세계관 DB·씬 개요·비트시트·AI 초안·시리즈 관리 흡수 경로 확정
- OfflineLocalStorage는 Phase D 클라이언트 레이어에서 재검토
- **후속**: V670 NolanAI 흡수 (G72-4)
