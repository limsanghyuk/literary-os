# ADR-079 — ConstitutionEvalV2: 헌법 멀티축 평가기 V2

**날짜**: 2026-05-26
**상태**: Accepted
**구현**: V637 (v11.7.0)

## 컨텍스트

SP-C.1 Self-Learning 파이프라인의 7번째 컴포넌트.
LOSConstitution 5축 가중합 점수를 다차원 채점 결과로 세분화하고,
인간 calibration(evaluator_id) 흔적을 추적 가능하게 설계.

## 결정

- **EvalDimension**: 평가 차원 정의 (dimension_id, weight)
- **EvalScore**: 단일 차원 채점 (raw_score × weight = weighted_score)
- **EvalResult**: 전체 결과 (final_score = Σ weighted_score, passed = final_score ≥ EVAL_THRESHOLD)
- **EVAL_THRESHOLD = 0.70**: 70% 이상이면 PASS
- **DEFAULT_DIMENSIONS**: coherence / authenticity / style_adherence / emotional_resonance / narrative_flow (각 0.2)
- **LOSDB JSONL append-only**: 메모리 + 파일 이중 모드
- **LLM-0 원칙**: 외부 LLM 호출 없음

## C-M-06 보강 (V632 미적용분)

`literary_system/finetune/lora_artifact.py`에 `constitution_weights_version: str` 필드 추가.
LoRA 아티팩트와 헌법 가중치 버전 간 추적 가능성 확보.
