# ADR-154: TraceSamplingController — OTel 샘플링 레이트 동적 제어 (V691)

## Status
Accepted — V691 (2026-05-28)

## Context
V688/V689로 OTel TraceContext와 Prometheus 통합을 완성했으나, 
모든 span을 샘플링하면 고부하 시 저장소 비용 급증 문제가 있다.

## Decision
`literary_system/ops/trace_sampler.py` 신설:
- `SamplingRate` — 0~1 frozen 값 객체
- `SamplingStrategy` — ALWAYS/NEVER/RATIO/ADAPTIVE
- `TraceSampler` — 전략별 결정 + 통계 수집  
- `AdaptiveSampler` — 에러율/P99 레이턴시 기반 자동 rate 조정
- 부모 TraceContext의 SAMPLED 플래그 상속 (tail sampling 방지)
