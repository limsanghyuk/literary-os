# ADR-120: FeedbackToRLHF Adapter — z-score 이상치 제거

## 상태
Accepted (V660, 2026-05-27)

## 맥락
ReaderFeedbackCollector(V659)가 수집한 피드백을 RLHF 파이프라인에 공급해야 한다.
이상치 점수(스팸·악의적 리뷰 등)가 학습에 개입되지 않도록 통계 필터링이 필요하다.

## 결정

### z-score 이상치 제거
- 기준: `|z| = |(score - mean) / std| > Z_THRESHOLD (2.0)`
- 전체 모표준편차(pstdev) 기준 → 샘플 크기 불일치 편향 방지
- 모든 점수 동일 시(std≈0) 이상치 없음으로 처리

### 정규화
- `normalized = (raw_score - 1.0) / 4.0` → 0.0~1.0
- 극단 점수(1.0, 5.0): 신뢰 가중치 0.85 (나머지 1.0)

### RLHFSample 필드
| 필드 | 설명 |
|------|------|
| sample_id | feedback_id 재사용 |
| normalized_score | 0.0~1.0 |
| weight | 신뢰 가중치 |
| meta.hashed_reader_id | 추적용 해시 (평문 없음) |

### LLM-0 준수
이 모듈은 외부 LLM API를 호출하지 않는다. 순수 통계 처리만 수행.
