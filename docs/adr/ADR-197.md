# ADR-197: FL PoC E2E 통합 — FedAvg + FLClient + Privacy (V735)

**날짜**: 2026-05-29
**버전**: v12.5.4 (V735)
**상태**: ACCEPTED
**결정자**: SP-D.4 FL 팀

## Context

V732~V734에서 federation/ 패키지의 핵심 4개 모듈이 완성되었다:
FLCoordinator, FedAvgAggregator, FLClient, FLPrivacyNoise.
V735는 이들을 하나의 E2E 연합 학습 루프로 통합 검증한다.

## Decision

E2E FL 루프:
1. FLCoordinator 초기화 (min_clients=2, max_rounds=5)
2. N개 FLClient 등록 (각 클라이언트 다른 ProseDataShard)
3. 매 라운드:
   a. coordinator.start_round()
   b. 각 클라이언트 client.receive_global_model(global_weights)
   c. 각 클라이언트 state = client.train(round_num)
   d. coordinator.submit_client_state(state)
   e. dp_weights = privacy.privatize(aggregated_weights)
   f. global_model = aggregator.aggregate(states, round_num)
   g. coordinator.finalize_round(global_model)
4. coordinator.is_converged() 또는 max_rounds 도달 시 종료

## Consequences

- G90 FL Gate FL-1~FL-5 검증 가능 (V736에서 구현)
- FL PoC TC ≥ 30 추가 (V735)
