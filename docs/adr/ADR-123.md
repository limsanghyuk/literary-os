# ADR-123: SDKStabilityGate G70 — PublicSDK 베타 안정성 게이트

**상태**: Accepted  
**날짜**: 2026-05-27  
**버전**: V663 / v11.36.0  
**관련**: ADR-116 (PublicSDK v1.0)

---

## 컨텍스트

SP-C.3 출시 전 PublicSDK가 실제 베타 사용자 20명 환경에서
P0 버그 없이 안정적으로 동작함을 검증해야 한다.

---

## 결정

`literary_system/gates/sdk_stability_gate.py`에 **SDKStabilityGate(G70)** 구현.

### 합격 조건

| 축 | 기준 | 근거 |
|----|------|------|
| P0 버그 | 0건 | 크래시·None 반환·예외 미처리 절대 불허 |
| 성공률 | 20/20 (100%) | 모든 베타 사용자 4메서드 전체 성공 |
| 평균 지연 | ≤ 5,000ms | offline_mode stub SLO |
| SDK 버전 | semver 형식 | `literary_system.sdk.__version__` |

### 베타 사용자 시뮬레이션

- 20명 × 4 메서드 (analyze / repair / predict / generate) = 80회 호출
- offline_mode=True (LLM 비용 없음)
- 텍스트 5종 × 컨텍스트 5종 순환

---

## 트레이드오프

| 항목 | 결정 |
|------|------|
| 실제 베타 vs 시뮬레이션 | 시뮬레이션 — CI 자동화 |
| SLO 5,000ms | offline stub은 <1ms, 여유 크게 줌 |
| P0 기준 | None 반환도 P0 처리 — 데이터 손실 방지 |
