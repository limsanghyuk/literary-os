# ADR-159: AgentMessage + AgentTask — 멀티에이전트 메시지·태스크 스키마 (V697)

**날짜**: 2026-05-28
**버전**: v12.1.1 (V697)
**상태**: ACCEPTED (소급)

## Context

멀티에이전트 시스템의 기초 데이터 구조로 에이전트 간 메시지 전달과 태스크
명세를 표준화해야 한다.

## Decision

`literary_system/agents/agent_message.py`:
- `AgentMessage(id, sender, receiver, content, timestamp, msg_type)`
- `serialize() / deserialize()` JSON 변환

`literary_system/agents/agent_task.py`:
- `AgentTask(task_id, description, priority, assigned_to, status, result)`
- 상태 FSM: PENDING → IN_PROGRESS → DONE / FAILED

## Consequences

- 에이전트 메시지 표준화로 CollaborationProtocol 기반 확보
- TaskScheduler가 AgentTask를 우선순위 큐로 관리
