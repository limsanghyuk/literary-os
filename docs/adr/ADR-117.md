# ADR-117: OpenAPI 3.1 Swagger + Postman + 3언어 샘플

## 상태
Accepted (V657, 2026-05-27)

## 맥락
V656에서 PublicSDK 코어를 구현했다. 외부 개발자·B2B 파트너가 SDK를 실제로 사용하려면
API 명세서, 테스트 도구(Postman), 언어별 샘플이 필요하다.

## 결정

### OpenAPI 3.1 스펙 자동 생성
`literary_system/sdk/api_schema.py` — `build_openapi_schema()` 호출로 dict 반환.
JSON/YAML 두 형식 모두 지원. PyYAML 미사용(내장 직렬화).

### 5개 엔드포인트
| 경로 | 메서드 | 역할 |
|------|--------|------|
| `/analyze` | POST | 씬 품질 5축 분석 |
| `/repair` | POST | 이슈 기반 수정 |
| `/predict` | POST | 다음 씬 예측 |
| `/generate` | POST | 씬 생성 |
| `/health` | GET | 헬스 체크 (인증 불필요) |

### 인증
- Bearer JWT (기본)
- X-API-Key 헤더 (대안)

### Postman Collection v2.1
`docs/sdk/postman_collection.json` — 5개 요청 + 환경변수(base_url, api_key).

### 3언어 샘플
| 파일 | 언어 |
|------|------|
| `docs/sdk/samples_python.py` | Python 3.10+ |
| `docs/sdk/samples_node.js` | Node.js 18+ (node-fetch) |
| `docs/sdk/samples_curl.sh` | cURL (bash) |

## 결과
- `literary_system/sdk/api_schema.py` 신설 (321줄)
- `docs/sdk/` 폴더 신설 (Postman + 3언어 샘플)
- TC 33개 추가 (test_v657_api_schema.py)
