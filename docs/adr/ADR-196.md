# ADR-196: FLClient + FLPrivacyNoise — 로컬 훈련 시뮬레이터 + DP (V734)

**날짜**: 2026-05-29
**버전**: v12.5.3 (V734)
**상태**: ACCEPTED
**결정자**: SP-D.4 FL 팀

## Context

V733의 FedAvgAggregator는 클라이언트 상태를 외부에서 주입받는 구조다.
V734에서는 클라이언트 측 로컬 훈련을 시뮬레이션하는 FLClient와
차분 프라이버시 노이즈 주입기 FLPrivacyNoise를 구현한다.

## Decision

`literary_system/federation/fl_client.py`:
- `ProseDataShard(shard_id, num_samples, base_loss, noise_std)` — 로컬 데이터 명세
- `FLClient(client_id, shard, learning_rate, local_epochs, weight_dim, seed)`
  - `receive_global_model(global_weights)` — 글로벌 가중치 수신
  - `train(round_num) → FLClientState` — local SGD N에폭 시뮬레이션
    - 손실: `base_loss * exp(-lr * epoch) + noise`
    - 가중치: `w -= lr * gradient_noise`

`literary_system/federation/fl_privacy.py`:
- `FLPrivacyNoise(epsilon, delta, clip_norm, seed)` — Gaussian Mechanism
  - `sigma = C * sqrt(2 * ln(1.25/δ)) / ε`
  - `clip_weights(weights)` — L2 노름 클리핑
  - `add_noise(weights)` — Gaussian 노이즈 주입
  - `privatize(weights)` — 클리핑 + 노이즈 파이프라인

## Rationale

FedAvg + DP-SGD(Abadi et al., 2016) 조합은 현재 연합 학습 프라이버시 보호의
표준이다. sigma 공식은 (ε, δ)-DP 보장을 수학적으로 제공한다.

## Consequences

- V735: FedAvg + FLClient + FLPrivacyNoise 통합 E2E 테스트 ≥30 TC
- G90 FL Gate FL-1~FL-5 검증 가능
