# ADR-225: RLAIFTrigger — GPU 학습 트리거 (LoRAJobRunner 연결)

## 상태
ACCEPTED (V765, Phase E.4)

## 맥락
loop-C DPO 데이터셋(V764)을 실 생성기 학습으로 연결. 로컬 환경 사양 조사 결과 **GPU 없음**(2 CPU·3.8GB·torch 미설치) → 실 학습은 클라우드 GPU만 가능.

## 결정
`learning/rlaif_trigger.py` RLAIFTrigger: RLAIFTrainingSpec → finetune.LoRAJobRunner.run(config, dataset, hours). 이 환경 dry_run(비용 추정만), 실 GPU=RunPod 주력/Lambda 폴백·GPU SLO(월 hard $120)·biweekly_train.yml. LLM-0.

## 결과
12 TC PASS. 개발자 17쌍 → dry_run 트리거(비용≈$0.78/2h). **E.4 RLAIF 사슬 GPU까지 연결**(실 학습은 클라우드). 미준비 스펙은 not_ready.
## 대안
sandbox 직접 학습: GPU 부재로 불가(기각). dry_run+클라우드 위임.
