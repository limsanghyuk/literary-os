# ADR-054: LOSConstitution v1.0 — 장면 품질 헌법

**상태**: 수락  
**날짜**: 2026-05-21  
**버전**: V594 (9.9.0)  
**Gate**: G51 ConstitutionGate  

---

## 맥락

Literary OS는 한국 드라마·소설 장면을 생성하는 AI 시스템이다. Phase A 종반부(SP-A.7)에서 생성된 장면의 품질을 **객관적·반복 가능한 방식**으로 평가하는 기준이 필요해졌다. 이 기준은 이후 RLHF(강화학습 인간 피드백) 보상 신호로도 활용된다.

기존 DRSE(Gate 7)는 단일 메트릭이었고, 산문 품질·이야기 구조·갈등 긴장도를 통합 평가하지 못했다.

---

## 결정

**LOSConstitution v1.0** — 5축 가중합 장면 품질 헌법을 도입한다.

### 5축 구성

| 축 | 가중치 | 측정 방법 | 임계값 |
|----|--------|-----------|--------|
| DRSE | 0.30 | length + TTR + narrative marker density | — |
| Debt | 0.20 | 미결 플롯훅 대비 해소 비율 | — |
| Arc | 0.20 | 기승전결 구조 신호 (4막 탐지) | — |
| Tension | 0.15 | 갈등/긴장 마커 밀도 | — |
| Prose | 0.15 | 어휘 다양성(TTR) + 문장 길이 변화 | — |

### 공식

```
R(scene) = 0.30·DRSE + 0.20·Debt + 0.20·Arc + 0.15·Tension + 0.15·Prose

W(work)  = mean(R_i) − 0.10·var(R_i)

R_rlhf   = clamp(R(generated) − R(original), −1.0, 1.0)
```

### 품질 기준 (Gate G51)

- **R(scene) 평균 ≥ 0.65**: 정상 드라마 장면 기준
- **variance ≤ 0.05**: 일관된 품질

## 구현

- `literary_system/constitution/los_constitution.py`

  - `LOSConstitution` — score_scene() / score_work() / rlhf_reward()
  - `ConstitutionSceneScore` / `ConstitutionWorkScore`
- **LLM-0 원칙**: 모든 메트릭은 순수 텍스트 통계 (외부 LLM 0건)

## 결과

**긍정**:
- 단일 수치로 장면 품질 비교 가능
- RLHF 보상 신호로 직접 활용
- 재현 가능 (동일 입력 → 동일 출력)

**부정/한계**:
- 문화적 뉘앙스(한국어 특유 표현) 일부 미반영
- 짧은 텍스트(< 20 단어) 신뢰도 낮음
