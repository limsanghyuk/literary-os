# ADR-185: ChaosEngine + FaultInjector (V724)

**날짜**: 2026-05-29 | **상태**: Accepted | **구현**: V724 (v12.3.8)

## 맥락

SP-D.3 G89 Chaos Resilience Gate 구현을 위해 장애 주입 엔진이 필요하다. chaos/ 패키지를 신설하고 V725 G88 Gate에서 import하여 ADR-128 고립을 해소한다.

## 결정

- **ChaosEngine**: FaultSpec 등록/활성화/주입/이력. FaultType 5종(NETWORK_PARTITION/MEMORY_PRESSURE/CPU_SPIKE/DISK_FULL/SERVICE_CRASH). 확률 기반 주입, 핸들러 콜백.
- **FaultInjector**: 함수 전/후 장애 주입 래퍼. inject_before/inject_after/wrap 데코레이터. 주입 이력(InjectorRecord) 자동 기록.
- chaos/ → gates/ 연결: V725 G88 Gate에서 import → 고립 해소
