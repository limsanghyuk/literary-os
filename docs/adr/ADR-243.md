# ADR-243: 정적-닻 자체평가 M1 — Critic 자격검정 (열화 사다리)

## 상태
ACCEPTED (V782) — 설계도 2026-06-17_static_anchor_self_eval_design_v1 구현(M1 먼저)

## 맥락
인간 작가 GT = 최종시험(Final Exam)으로 보류. 그때까지 명작 원본(정적 닻)으로 인간 없이 자체평가해 LLM↔LLM 순환 돌파. alignment_monitor의 닻(인간 GT)이 비어 있음 → M1이 부분 대체.

## 결정
`critic/critic_qualification.py`:
- `degrade(text, axis, severity)`: 축별 표적 열화(EMOTION 감정평탄/FORESHADOW 복선제거/CAUSALITY 인과단절/DICTION 상투어휘), 등급화(severity 0~1). 결정론(LLM-0).
- `build_ladder`: severity 사다리(0.25~1.0).
- `qualify_critic(judge, masterpieces)`: judge가 명작>열화판을 (a) 전구간 우위(win≥0.80) (b) 단조 변별곡선으로 가르는지 검정 → 둘 다 충족 critic만 자체평가 심판 자격.

## 결과
14 TC PASS. 좋은 critic 자격통과(승률 0.84·단조)/무능·역전 critic 미달. **축별 열화로 critic의 축별 판단력 노출**(예: 길이/키워드 judge는 break_causality 축 평탄=인과 변별 약함). = 표면 흔적 부정통과 차단·변별 해상도 측정.
**천장 정직**: M1은 자격검정(열화 판별)일 뿐 신작 *생성* 판정 자격 아님. 독창성은 인간 최종시험 몫. 다음=M2 NextEpisodeBench(은닉GT)·M3 분포 음성가드.
## 대안
단일 픽 판별: 변별 해상도 측정 불가(기각). 등급화 사다리+단조성으로 해상도 확보.
