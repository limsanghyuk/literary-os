## 맥락

Phase D 본안(D-M-11)에서 G91 DR Gate 신설을 명시:
"V741~V743, BackupManager, DR-1~DR-5, RPO≤1h, ≥20 TC".

ADR-201(Phase E Manifest Validator)과 동일한 패턴으로
5개 체크함수 + `run_dr_gate()` 통합 실행기를 구현한다.

## 결정

`literary_system/gates/dr_gate.py`에 5개 체크(DR-1~DR-5) + `run_dr_gate()` 구현.

### 체크 명세

| ID | 대상 | 검증 내용 |
|----|------|-----------|
| DR-1 | DRBackupManager | `interval=3601` → ValueError, `interval=3600` → OK |
| DR-2 | DRBackupManager | `create_backup()` → SUCCESS + SHA-256 64자 |
| DR-3 | DRBackupManager | `rpo_compliant()` True→False 경계(3600s) |
| DR-4 | DRRestoreManager | 정확 데이터 → SUCCESS, 변조 → CHECKSUM_MISMATCH |
| DR-5 | E2E Pipeline | 4백업 → 프루닝(≤3), 테넌트 격리, 복원 SUCCESS |

### Gate 통과 조건

- 5개 체크 모두 PASS
- `GATES.append(("dr_g91", ..., _gate_dr_g91))` 등록

## 결과

- GATES 총 93개 (G91 추가 후)
- Phase D Exit Gate G95에서 G91 PASS 여부를 재확인
