# ADR-045: LOSDBClient — LOSDB 통합 Facade

**상태**: 승인  
**버전**: V586 (9.1.0)  
**날짜**: 2026-05-20  
**레이어**: L1  

---

## 컨텍스트

V585로 LOSDB Phase B (SQLiteRealAdapter / VectorRealAdapter / GraphRealAdapter) 가 완료되었다. 그러나 세 어댑터는 각자 독립적으로 존재하며 통합 진입점이 없어 사용자 학습 비용이 높다.

## 결정

`LOSDBClient` Facade 클래스를 `literary_system/db/losdb_client.py`에 신설한다.

- 세 어댑터를 옵셔널 파라미터로 받아 통합 관리
- `available_backends()`, `query_by_label()`, `cross_query()`, `check_all_connections()`, `schema_info()` 제공
- `LOSDBClientRecord(id, backend, label, metadata)` 통합 결과 타입

## 결과

- Gate G45 (ADR-045, L1) 신설 — 8체크포인트
- 버전 9.1.0 (Phase C 시작 마이너 범프)
- 기존 어댑터 변경 없음 (호환성 유지)
