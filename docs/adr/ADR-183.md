# ADR-183: AgentAuthBridge — agents/security 단방향 통합 (V722)

**날짜**: 2026-05-29  
**상태**: Accepted  
**구현**: V722 (v12.3.6)

## 맥락

V721에서 plugins/ → security/ 연결로 security 고립을 해소했다. V722에서는 에이전트 메시지 발송 전 ZeroTrust 검증을 강제하여 에이전트 간 신뢰 경계를 확립한다.

## 결정

1. **AgentAuthBridge** 신설(`literary_system/agents/agent_auth_bridge.py`)
   - `agents/ → security/` 단방향 의존
   - 에이전트별 tenant_id + required_role 등록
   - `check(agent_id, token_str) → BridgeResult(decision, claims, reason)`
   - `fail_closed=True`: 미등록 에이전트도 DENY
   - 감사 로그 + 통계 내장

## 결과

- 에이전트 메시지 Zero-Trust 인증 강제화
- agents/ → security/ 연결 → security 패키지 2경로 연결 (plugins/ + agents/)
- 33 TC PASS, G32 0 violations
