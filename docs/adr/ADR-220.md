# ADR-220: 심판 상향 — 축별 3페르소나 합의 critic

## 상태
ACCEPTED (V760, Phase E.2)

## 맥락
(b) GPT-5 비교 실측에서 단일 gpt-4o-mini 심판이 두 드래프트를 전 축 TIE로 변별 못 함(V755 잔여 항목 ⓑ).

## 결정
LLMCritic에 `n_judges`(기본1)·페르소나(문학평론가·드라마투르그·일반시청자) 추가. n_judges>1이면 페르소나별 독립 블라인드 판정 → 다수결 합의(동률 tie). make_ensemble/CriticEnsemble로 전파. 심판 모델은 주입형(gpt-5 선택 가능).

## 근거
다수 페르소나 합의가 단일 판정보다 강건(1/3 반대 흡수 실증). 기본 n_judges=1로 하위호환.
## 결과
14 TC PASS, critic 회귀 무파손. V755 잔여 해소.
## 대안
단일 심판 유지: 변별력 부족(기각).
