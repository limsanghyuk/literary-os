# ADR-219: Arbitration Protocol v1 (공식↔critic 중재)

## 상태
ACCEPTED (V759, Phase E.2)

## 맥락
공식(구조 게이트)과 LLM critic이 불일치할 때 처리 규칙이 없었다. 불일치를 학습신호로 전환한다.

## 결정
`critic/arbitration.py`: 공식 winner(LOSConstitution R)↔critic winner 비교 → 5분기(agree/formula_defect/critic_defect/genuine_ambiguous/pending). pending은 disagreement_queue(인간 큐)로. 인간 판정이 공식 recalibrate / critic 개선 신호.

## 결과
16 TC PASS. Phase E.2의 LLM-0↔LLM-1 충돌 해소 메커니즘. 인간 GT(human_gt)·공식 생애주기(ledger)와 연결.
## 대안
불일치 무시: 학습신호 손실(기각).
