# ADR-170: SP-D.2 통합 테스트 스위트 — MultiAgent Coordination 전체 검증 (V708~V709)

**날짜**: 2026-05-28
**버전**: v12.2.2 (V708~V709)
**상태**: ACCEPTED (소급)

## Context

SP-D.2 전체 모듈(ADR-158~169)에 대한 통합 회귀 테스트가 단일
테스트 파일로 정리되어야 한다.

## Decision

`tests/unit/test_v706_v709_spd2_integration.py`:
- 6축 × 2게이트 = 12 핵심 TC
- AgentCoordinationGate 단독 실행
- AgentWorkflowGate 단독 실행
- 전체 에이전트 파이프라인 E2E 시나리오
- CollaborationProtocol + ConflictResolver 조합 테스트

SP-D.2 Exit Gate(G85-EXIT, ADR-171)가 모든 게이트 누적 결과를 종합.

## Consequences

- SP-D.2 모든 모듈 커버리지 확보
- ADR-171(SP-D.2 Exit Gate) 진입 기준 충족
