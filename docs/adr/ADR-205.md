# ADR-205: G92 Phase D Performance SLO Gate

**상태**: Accepted  
**날짜**: 2026-05-29  
**버전**: V744  
**작성자**: Literary OS Team  

## 컨텍스트

Phase D SP-D.4 보조 게이트 시리즈의 첫 번째 게이트. Phase D에서 도입된 AgentBus, CircuitBreaker, DRBackupManager 등 성능 임계값(SLO) 준수 여부를 자동 검증해야 한다.

설계도(Phase D Blueprint) 의거 SLO 기준:
- REST API P99 ≤ 200ms
- AgentBus RTT ≤ 50ms
- DRBackup 생성 지연 ≤ 100ms

## 결정

G92 Performance SLO Gate를 `literary_system/gates/spd4_aux_gates.py`에 구현한다.

| 체크 | 검증 대상 | 합격 조건 |
|------|-----------|-----------|
| PS-1 | AgentBus 모듈 임포트 | `literary_system.agents.agent_message.AgentBus` 클래스 존재 |
| PS-2 | AgentCircuitBreaker 상태 전환 | CLOSED → OPEN 상태 전환 로직 동작 |
| PS-3 | SLO 상수 검증 | `API_P99_SLO_MS ≤ 200.0`, `AGENT_RTT_SLO_MS ≤ 50.0` |
| PS-4 | P99 분위수 정확도 | `statistics.quantiles` 기반 P99 ∈ [98.0, 100.0] ms |
| PS-5 | DRBackup 생성 지연 | 실제 백업 생성 시간 ≤ 100ms |

## 근거

- AgentCircuitBreaker는 `literary_system.agents.circuit_breaker`에 위치하며 `CircuitBreakerConfig(failure_threshold, timeout_seconds)` 패턴을 사용
- statistics.quantiles는 TD-1 SLO 수식에서 표준으로 채택된 분위수 방법
- LLM-0 원칙: 모든 검증은 로컬 코드 실행, 외부 LLM 호출 없음

## 결과

G92 PASS 시 Phase D 성능 SLO 준수 자동 증명. release_gate.py GATES 목록에 등록.
