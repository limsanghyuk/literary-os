# ADR-156: SP-D.1 통합 테스트 스위트 — 크로스 모듈 관측성 검증

**Date:** 2026-05-28
**Status:** Accepted

## Context

V688~V692에서 구축된 5개 관측성 모듈(TraceContext, OtelAdapter, PrometheusTraceExtension,
TraceSampler, ObservabilityDashboard)의 크로스 모듈 연동을 검증하는 통합 테스트가 없었다.
각 모듈은 개별 단위 테스트가 있었으나, 모듈 간 API 호환성(실제 파라미터명, 반환 타입 등)은
별도로 검증되지 않았다.

## Decision

`tests/integration/test_v693_spd1_observability_integration.py` 에 33 TC 통합 테스트를 구현한다.

| 섹션 | TC | 검증 내용 |
|------|----|----------|
| TC-01~06 | 6 | TraceContext + OtelSdkAdapter 연동 |
| TC-07~12 | 6 | TraceSampler + OtelSdkAdapter 파이프라인 |
| TC-13~18 | 6 | PrometheusTraceExtension E2E |
| TC-19~24 | 6 | ObservabilityDashboard + OTel 메트릭 통합 |
| TC-25~30 | 6 | AdaptiveSampler + Dashboard 피드백 루프 |
| TC-31~33 | 3 | 전체 스택 E2E 시나리오 |

## Consequences

- 크로스 모듈 API 호환성이 CI에서 지속 검증됨
- 실제 API 시그니처 기반으로 테스트 작성 → 허상 테스트 제거
- 누적 TC: 9,205 (9,172 + 33)
