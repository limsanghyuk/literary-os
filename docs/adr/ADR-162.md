# ADR-162: ConflictResolver — 에이전트 충돌 해결기 (V700)

**날짜**: 2026-05-28
**버전**: v12.1.4 (V700)
**상태**: ACCEPTED (소급)

## Context

두 에이전트가 동일한 씬 노드를 동시에 수정하면 충돌이 발생한다.
우선순위·타임스탬프·역할 기반 해결 전략이 필요하다.

## Decision

`literary_system/agents/conflict_resolver.py`:
- `ConflictResolver` 3-way 전략: PRIORITY_WINS / TIMESTAMP_WINS / ROLE_WINS
- `resolve(conflict: Conflict) → Resolution`
- 해결 불가 시 `EscalationException` 발생 → 감독자(AgentSupervisor) 에스컬레이션

## Consequences

- G84 체크포인트 E4(ConflictResolver) 충족
- 동시 편집 충돌 자동 해결
