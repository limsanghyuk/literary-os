# ADR-161: CollaborationProtocol — 에이전트 협업 프로토콜 (V699)

**날짜**: 2026-05-28
**버전**: v12.1.3 (V699)
**상태**: ACCEPTED (소급)

## Context

멀티에이전트 환경에서 두 에이전트가 공동 태스크를 수행할 때 표준 협업
절차가 없으면 데드락과 중복 실행이 발생한다.

## Decision

`literary_system/agents/collaboration_protocol.py`:
- `CollaborationSession(session_id, initiator, participants, shared_context)`
- `propose(agent_id, action) → bool` — 참여자 과반 승인 시 True
- `commit(session_id)` / `rollback(session_id)` — 트랜잭션 의미론

## Consequences

- G84 체크포인트 E3(CollaborationProtocol) 충족
- 에이전트 데드락 방지
