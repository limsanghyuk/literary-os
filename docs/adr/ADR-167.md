# ADR-167: agents/CircuitBreaker — 에이전트 회로 차단기 (V705)

**날짜**: 2026-05-28
**버전**: v12.1.9 (V705)
**상태**: ACCEPTED (소급)

## Context

에이전트가 반복 실패하면 전체 파이프라인이 블로킹될 수 있다.
회로 차단기로 빠른 실패(Fast Fail)를 보장한다.

## Decision

`literary_system/agents/circuit_breaker.py`:
- `AgentCircuitBreaker(agent_id, threshold=3, timeout_s=60)`
- 상태: CLOSED → OPEN (threshold 실패) → HALF_OPEN (timeout 후) → CLOSED
- `call(fn, *args) → Any` — OPEN 시 즉시 `CircuitOpenError`

## Consequences

- G85 체크포인트 E4(CircuitBreaker) 충족
- 장애 전파 차단
