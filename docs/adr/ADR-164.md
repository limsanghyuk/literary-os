# ADR-164: AgentWorkflow — 에이전트 워크플로우 오케스트레이터 (V702)

**날짜**: 2026-05-28
**버전**: v12.1.6 (V702)
**상태**: ACCEPTED (소급)

## Context

멀티에이전트 협업에서 에이전트 실행 순서와 데이터 흐름을 DAG 형태로
정의할 수 있어야 한다.

## Decision

`literary_system/agents/agent_workflow.py`:
- `AgentWorkflow(steps: List[WorkflowStep])` — DAG 노드 순서
- `WorkflowStep(agent_id, input_keys, output_keys, condition)`
- `execute(context: dict) → dict` — 단계별 순차 실행, 조건부 분기

## Consequences

- G85 체크포인트 E1(AgentWorkflow) 충족
- 복잡한 다단계 에이전트 파이프라인 정의 가능
