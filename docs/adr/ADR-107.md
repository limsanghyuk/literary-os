# ADR-107: ScriptAgent — LoRA InferenceGateway 직결 초안 생성 (V647)

## 상태
Accepted

## 버전
V647 (v11.18.0)

## 날짜
2026-05-26

## 맥락
SP-C.2 DirectorAgent(V646) 이후 초안 생성 단계. LoRA InferenceGateway(V598)에 직결하여
씬 텍스트를 생성. AgentSafetyGuard(V653)가 미완성이므로 사전 검사는 Guard 없으면 통과.

## 결정
- ScriptAgent: LoRA InferenceGateway 경유 (LLM-0/LLM-1 준수)
- HAS_LORA=False 환경 → StubBackend로 자동 fallback (테스트 안전)
- C-M-09: attempt_num > 3 → ValueError (AgentCoordinator가 강제)
- 사전 SafetyGuard 검사: Guard 주입 없으면 기본 통과

## 근거
LLM-0/LLM-1 원칙 준수. 외부 API 직접 호출 없음. PROMOTED LoRA 아티팩트만 사용.

## 결과
- literary_system/agents/script_agent.py 신설
- ScriptDraft 데이터클래스 (scene_id, draft_text, attempt_num, safety_passed, lora_artifact_id, word_count)
- test_v647_script_agent.py 30 TC

## 대안 검토
