# ADR-247: 클라우드 비공개 저장 + 실측 학습 노드 (저작권 안전)

## 상태
ACCEPTED (V786) — 회사 핸드오프 §3.1 미구현 보강 구현. V777 라이프사이클 위.

## 맥락
회사 PC=노트북 약GPU → 클라우드 학습 권장. 단 선호쌍은 verbatim(명작 원문) 포함 → 공개 저장 금지. 비공개·암호화·임시 저장 + 자동삭제(저작권 안전) + 학습 노드 마무리 필요.

## 결정
- `finetune/cloud_storage.py`: CloudStore 계약 + PresignedHttpStore(프리사인 S3/B2 호환, 클라이언트 SHA256-CTR 암호화, 업로드 추적·cleanup 자동삭제). encrypt/decrypt_bytes(대칭). ※운영 AEAD(Fernet/age) 권장 명시.
- `finetune/cloud_training_node.py` CloudTrainingNode: 선호쌍→암호화 업로드→RunPod QLoRA DPO(V777 라이프사이클)→어댑터 회수→**자동삭제**→ΔW→G_LOOPC_WINRATE 수용판정.
- 주입형 transport로 자격증명 없이 테스트(실 스토리지 엔드투엔드는 사용자 키 필요).

## 결과
15 TC PASS. 암호화 왕복·verbatim 미노출·자동삭제(잔존0)·ΔW·수용판정 검증. 집(4070 로컬)↔회사(클라우드 노드) 대칭 학습 노드 완성. dry_run/키없음=계획만(안전). LLM-0.
## 대안
평문 클라우드 저장: 저작권 위험(기각). 비공개 암호화+자동삭제 채택.
